# <a name="set-up-subscription-groups"></a><span data-ttu-id="1b5f1-103">إعداد مجموعات الاشتراك</span><span class="sxs-lookup"><span data-stu-id="1b5f1-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="1b5f1-104">انقر فوق **إدارة الخدمة** \> **إعداد** \> **اشتراكات الخدمة** \> **مجموعات الاشتراك**.</span><span class="sxs-lookup"><span data-stu-id="1b5f1-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="1b5f1-105">اضغط على CTRL+N أو انقر فوق **جديد**  لإنشاء مجموعة اشتراك جديدة.</span><span class="sxs-lookup"><span data-stu-id="1b5f1-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="1b5f1-106">في حقل **مجموعة الاشتراك**، أدخل اسمًا لمجموعة الاشتراك الجديدة.</span><span class="sxs-lookup"><span data-stu-id="1b5f1-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="1b5f1-107">في حقل **الفوترة**، حدد معدل تكرار الفوترة للاشتراك.</span><span class="sxs-lookup"><span data-stu-id="1b5f1-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="1b5f1-108">تتم الإشارة إلى معدل تكرار الفوترة بكود فترة والذي يتم إعداده في نموذج **أنواع الفترات**.</span><span class="sxs-lookup"><span data-stu-id="1b5f1-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="1b5f1-109">لاستحقاق الإيراد لحركات الاشتراكات، حدد خانة الاختيار **الإيراد المستحق**، ثم في حقل **الاستحقاق**، حدد تكرار.</span><span class="sxs-lookup"><span data-stu-id="1b5f1-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="1b5f1-110">يعتمد معدل تكرار الاستحقاق على كود فترة يتم إعداده في نموذج **الفترات**.</span><span class="sxs-lookup"><span data-stu-id="1b5f1-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
