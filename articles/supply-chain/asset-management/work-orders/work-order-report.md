<span data-ttu-id="44a22-104">يمكنك إنشاء تقرير أمر عمل يوضح المعلومات التفصيلية حول أمر عمل واحد حددته أو أكثر.</span><span class="sxs-lookup"><span data-stu-id="44a22-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>
