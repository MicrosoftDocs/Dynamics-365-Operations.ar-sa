<span data-ttu-id="785ce-104">يمكنك إنشاء تقرير أمر عمل يوضح المعلومات التفصيلية حول أمر عمل واحد حددته أو أكثر.</span><span class="sxs-lookup"><span data-stu-id="785ce-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

2. <span data-ttu-id="785ce-106">حدد "نعم"/"لا" على أزرار التبديل كما هو مطلوب، لتحديد التفاصيل التي سيتم تضمينها في التقرير.</span><span class="sxs-lookup"><span data-stu-id="785ce-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>
