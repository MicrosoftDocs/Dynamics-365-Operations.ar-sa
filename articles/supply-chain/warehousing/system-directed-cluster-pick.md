# <a name="system-directed-cluster-picking"></a><span data-ttu-id="acb92-103">انتقاء نظام مجموعة بتوجيه من النظام</span><span class="sxs-lookup"><span data-stu-id="acb92-103">System-directed cluster picking</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="acb92-104">انتقاء نظام المجموعة هو عملية انتقاء قطعة تسمح لك بانتقاء الأصناف لأوامر متعددة في الوقت نفسه عن طريق تجميعها في أنظمة مجموعة الانتقاء.</span><span class="sxs-lookup"><span data-stu-id="acb92-104">Cluster picking is a piece picking process that lets you pick items for multiple orders at the same time by clustering them into pick clusters.</span></span> <span data-ttu-id="acb92-105">ثم يجب عليك زيارة موقع الانتقاء مرة واحدة فقط.</span><span class="sxs-lookup"><span data-stu-id="acb92-105">You then have to visit the pick location only one time.</span></span> <span data-ttu-id="acb92-106">عادة، يتم استخدام هذه الوظيفة مع انتقاء أمر صغير أو كميات أقل من كميات الحالة.</span><span class="sxs-lookup"><span data-stu-id="acb92-106">Typically, this functionality is used with small order picking or quantities that are less than case quantities.</span></span>

<span data-ttu-id="acb92-107">عند إعداد انتقاء نظام المجموعة الموجه بواسطة النظام، فإنه يمكنك انتقاء رؤوس العمل الخاصة بنظام المجموعة، استنادًا إلى نظام المجموعة تم إنشاؤه بواسطة النظام.</span><span class="sxs-lookup"><span data-stu-id="acb92-107">When system-directed cluster picking is set up, you can cluster-pick work headers, based on a system-generated cluster.</span></span> <span data-ttu-id="acb92-108">تصل أوامر انتقاء نظام مجموعة النظام إلى عدد المناصب الذي تم تحديده في ملف تعريف نظام المجموعة.</span><span class="sxs-lookup"><span data-stu-id="acb92-108">The system cluster-picks orders up to the number of positions that is specified in the cluster profile.</span></span> <span data-ttu-id="acb92-109">لذلك، يمكنك انتقاء أوامر متعددة في الوقت نفسه دون الاضطرار إلى إنشاء نظام مجموعة يدويًا.</span><span class="sxs-lookup"><span data-stu-id="acb92-109">Therefore, you can pick multiple orders at the same time without having to manually create a cluster.</span></span>

<span data-ttu-id="acb92-110">يقدم انتقاء نظام المجموعة الموجه بواسطة النظام إنشاء نظام مجموعة يدوي، حيث يتم استخدام ملف تعريف نظام المجموعة لإنشاء نظام مجموعة.</span><span class="sxs-lookup"><span data-stu-id="acb92-110">System-directed cluster picking offers an alternative to manual cluster building, where a cluster profile is used to create a cluster.</span></span> <span data-ttu-id="acb92-111">يجب تعريف السطور ذات الصلة بالإعداد المتعددة في ملف تعريف نظام المجموعة قبل استخدامها:</span><span class="sxs-lookup"><span data-stu-id="acb92-111">Several setup-related lines must be defined in the cluster profile before it's used:</span></span>

- <span data-ttu-id="acb92-116">يحدد **فرز نوع التحقق** ما إذا كان التحقق من صحة المنصب مطلوبًا أم لا.</span><span class="sxs-lookup"><span data-stu-id="acb92-116">**Sort verification type** determines whether position verification is required.</span></span>

<span data-ttu-id="acb92-117">يتم استخدام صنف قائمة جهاز محمول جديد لانتقاء نظام المجموعة الموجه بواسطة النظام.</span><span class="sxs-lookup"><span data-stu-id="acb92-117">A new mobile device menu item is used for system-directed cluster picking.</span></span> <span data-ttu-id="acb92-118">يجب أن يتم تحديد معرف نظام المجموعة للخيار **الموجه بواسطة**.</span><span class="sxs-lookup"><span data-stu-id="acb92-118">The cluster profile ID must be specified for the **Directed by** option.</span></span> <span data-ttu-id="acb92-119">بالإضافة إلى ذلك، يمكن لأمر الاستعلام الموجه بواسطة النظام تجميع الأوامر، استنادًا إلى معايير خاصة بالشركة.</span><span class="sxs-lookup"><span data-stu-id="acb92-119">Additionally, the system-directed query order can group orders, based on company-specific criteria.</span></span> <span data-ttu-id="acb92-120">لذلك، يمكنك زيادة تحسين تعيين أوامر العمل عن طريق تحديد معايير الفرز المخصصة في أمر الاستعلام الموجه من قبل النظام.</span><span class="sxs-lookup"><span data-stu-id="acb92-120">Therefore, you can further optimize the assignment of work orders by specifying customized sorting criteria on the system-directed query order.</span></span>

<span data-ttu-id="acb92-121">عند إجراء انتقاء نظام المجموعة الموجه بواسطة النظام، فإنه يتم تقديم عمال المستودعات بنظام المجموعة حيث يتم إعادة تعيين أوامر الانتقاء إلى مناصب نظام المجموعة.</span><span class="sxs-lookup"><span data-stu-id="acb92-121">When system-directed cluster picking is done, warehouse workers are presented with a cluster where picking orders have been preassigned to cluster positions.</span></span> <span data-ttu-id="acb92-122">ولذلك، يمكن للعمال البدء في انتقاء صنف لأوامر عمل متعددة عن طريق زيارة موقع الانتقاء مرة واحدة فقط.</span><span class="sxs-lookup"><span data-stu-id="acb92-122">Therefore, workers can start to pick an item for multiple work orders by visiting the pick location only one time.</span></span> <span data-ttu-id="acb92-123">تُعد عملية الانتقاء لانتقاء نظام مجموعة موجه بواسطة النظام هي نفسها العميلة الخاصة بانتقاء نظام مجموعة موجه من قبل المستخدم.</span><span class="sxs-lookup"><span data-stu-id="acb92-123">The picking process for system-directed cluster picking is the same as the process for user-directed cluster picking.</span></span>
