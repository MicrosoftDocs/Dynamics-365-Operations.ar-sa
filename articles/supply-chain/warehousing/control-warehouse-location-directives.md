## <a name="work-templates"></a><span data-ttu-id="c9c99-108">قوالب العمل</span><span class="sxs-lookup"><span data-stu-id="c9c99-108">Work templates</span></span>

<span data-ttu-id="c9c99-109">تتيح لك صفحة **قوالب العمل** تحديد عمليات العمل التي يجب القيام بها في المستودع.</span><span class="sxs-lookup"><span data-stu-id="c9c99-109">The **Work templates** page lets you define the work operations that must be performed in the warehouse.</span></span> <span data-ttu-id="c9c99-110">وبشكل عام، تتكون عمليات عمل المستودع من زوج من الإجراءات: عامل مستودع يقوم بانتقاء المخزون الفعلي في موقع واحد، ثم يضع المخزون المنتقى في موقع آخر.</span><span class="sxs-lookup"><span data-stu-id="c9c99-110">Typically, warehouse work operations consist of a pair of actions: a warehouse worker picks up on-hand inventory in one location and then puts the picked inventory down in another location.</span></span>

<span data-ttu-id="c9c99-116">ويمكن استخدام الإعدادات الموجودة في تعريف رأس العمل لتحديد متى يجب إنشاء قطعة عمل جديدة.</span><span class="sxs-lookup"><span data-stu-id="c9c99-116">The settings in the work header definition can be used to determine when a new piece of work should be created.</span></span> <span data-ttu-id="c9c99-117">على سبيل المثال، يمكنك تعيين الحد الأقصى لعدد بنود الانتقاء وأقصى وقت متوقع للانتقاء.</span><span class="sxs-lookup"><span data-stu-id="c9c99-117">For example, you can set a maximum number of pick lines and a maximum expected pick time.</span></span> <span data-ttu-id="c9c99-118">وبعد ذلك، إذا تجاوز العمل لأي عملية انتقاء أمر مبيعات أي قيمة من هذه القيم، فإنه يتم تقسيم العمل إلى قطعتي عمل.</span><span class="sxs-lookup"><span data-stu-id="c9c99-118">Then, if the work for a sales order picking process exceeds either of those values, that work is split into two pieces of work.</span></span>

<span data-ttu-id="c9c99-124">ويمكنك تعيين استعلام للتحكم عند استخدام قالب عمل معين.</span><span class="sxs-lookup"><span data-stu-id="c9c99-124">You can set up a query to control when a particular work template is used.</span></span> <span data-ttu-id="c9c99-125">على سبيل المثال، يمكنك تعيين قيد حتى يمكن استخدام قالب معين للعمل في مستودع معين فقط.</span><span class="sxs-lookup"><span data-stu-id="c9c99-125">For example, you can set a limitation so that a particular template can be used for work only in a specific warehouse.</span></span> <span data-ttu-id="c9c99-126">وبدلاً من ذلك، قد يكون لديك العديد من القوالب التي يتم استخدامها لإنشاء عمل لأمر المبيعات الصادرة، استناداً إلى أصل المبيعات.</span><span class="sxs-lookup"><span data-stu-id="c9c99-126">Alternatively, you might have several templates that are used to create work for outbound sales order processing, depending on the sales origin.</span></span> <span data-ttu-id="c9c99-127">يستخدم النظام الحقل **الرقم التسلسلي‬** لتحديد الترتيب الذي يتم به تقييم قوالب العمل المتوفرة.</span><span class="sxs-lookup"><span data-stu-id="c9c99-127">The system uses the **Sequence number** field to determine the order that the available work templates are assessed in.</span></span> <span data-ttu-id="c9c99-128">ولذلك، إذا كان لديك استعلام محدد جداً لقالب عمل معين، يجب أن تعطيه رقمًا تسلسليًا منخفضًا.‬</span><span class="sxs-lookup"><span data-stu-id="c9c99-128">Therefore, if you have a very specific query for a particular work template, you should give it a low sequence number.</span></span> <span data-ttu-id="c9c99-129">وبعد ذلك، يتم تقييم هذا الاستعلام قبل استعلامات عامة أخرى كثيرة.</span><span class="sxs-lookup"><span data-stu-id="c9c99-129">That query will then be evaluated before other, more general queries.</span></span> 

<span data-ttu-id="c9c99-130">ولإيقاف عملية عمل أو إيقافها مؤقتًا، يمكنك استخدام إعداد **إيقاف العمل** في بند العمل.</span><span class="sxs-lookup"><span data-stu-id="c9c99-130">To stop or pause a work process, you can use the **Stop work** setting on the work line.</span></span> <span data-ttu-id="c9c99-131">وفي هذه الحالة، لن تتم مطالبة العامل الذي يقوم بإجراء العمل بتنفيذ خطوة بند العمل التالية.</span><span class="sxs-lookup"><span data-stu-id="c9c99-131">In that case, the worker who is performing the work won't be asked to perform the next work line step.</span></span> <span data-ttu-id="c9c99-132">وللانتقال إلى الخطوة التالية، يجب على هذا العامل أو عامل آخر تحديد العمل مرة أخرى.</span><span class="sxs-lookup"><span data-stu-id="c9c99-132">To move on to the next step, that worker or another worker must select the work again.</span></span> <span data-ttu-id="c9c99-133">كما يمكنك فصل المهام في جزء من العمل باستخدام *معرف فئة عمل* مختلف في بنود قالب العمل.</span><span class="sxs-lookup"><span data-stu-id="c9c99-133">You can also separate the tasks within a piece of work by using a different *work class ID* on the work template lines.</span></span>

## <a name="location-directives"></a><span data-ttu-id="c9c99-134">توجيهات الموقع</span><span class="sxs-lookup"><span data-stu-id="c9c99-134">Location directives</span></span>

<span data-ttu-id="c9c99-135">توجيهات الموقع هي القواعد التي تساعد في تحديد انتقاء ووضع مواقع لحركة المخزون.</span><span class="sxs-lookup"><span data-stu-id="c9c99-135">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="c9c99-136">على سبيل المثال، في حركة أمر مبيعات، يحدد توجيه الموقع أين سيتم انتقاء الأصناف وأين سيتم وضع الأصناف المنتقاة.</span><span class="sxs-lookup"><span data-stu-id="c9c99-136">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="c9c99-137">وتوجيهات الموقع تتكون من رأس والبنود المقترنة به، ويمكنك إنشاؤها في صفحة **توجيهات الموقع**.</span><span class="sxs-lookup"><span data-stu-id="c9c99-137">Location directives consist of a header and associated lines, and you create them on the **Location directives** page.</span></span> 

<span data-ttu-id="c9c99-138">وفي الرأس، يجب إقران كل توجيه موقع *بنوع أمر العمل* الذي يحدد نوع حركة المخزون التي سيتم استخدامه التوجيه لها، مثل أوامر المبيعات أو التزويد أو انتقاء المواد الخام.</span><span class="sxs-lookup"><span data-stu-id="c9c99-138">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="c9c99-139">ويحدد *نوع العمل* ما إذا كان سيتم استخدام توجيه الموقع لانتقاء ووضع العمل، أو لعمليات مستودع أخرى، مثل تغيير حالة الجرد أو المخزون.</span><span class="sxs-lookup"><span data-stu-id="c9c99-139">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="c9c99-140">كما يجب عليك تحديد *موقع* و *مستودع*.</span><span class="sxs-lookup"><span data-stu-id="c9c99-140">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="c9c99-141">ويمكن استخدام *كود توجيه* تقوم بتحديده في الرأس لربط توجيه الموقع بقالب عمل واحد أو أكثر.</span><span class="sxs-lookup"><span data-stu-id="c9c99-141">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span> 

<span data-ttu-id="c9c99-142">أما بالنسبة لقوالب العمل، يمكنك إعداد استعلام لتحديد وقت استخدام توجيه موقع معين.</span><span class="sxs-lookup"><span data-stu-id="c9c99-142">As for work templates, you can set up a query to determine when a particular location directive is used.</span></span> <span data-ttu-id="c9c99-143">على سبيل المثال، يمكنك تحديد أنه عندما التجارة الإلكترونية عي أصل أمر المبيعات، فيجب انتقاء المخزون من منطقة مخصصة في المستودع.</span><span class="sxs-lookup"><span data-stu-id="c9c99-143">For example, you can specify that when e-commerce is the origin of a sales order, the inventory must be picked up from a dedicated area in the warehouse.</span></span> <span data-ttu-id="c9c99-144">يستخدم النظام الحقل **رقم التسلسلي‬** لتحديد الترتيب الذي يتم به تقييم توجيهات الموقع المتوفر.</span><span class="sxs-lookup"><span data-stu-id="c9c99-144">The system uses the **Sequence number** field to determine the order that the available location directives are assessed in.</span></span>

<span data-ttu-id="c9c99-145">وتقوم بنود توجيهات الموقع بتعيين قيود إضافية على تطبيق قواعد البحث عن الموقع.</span><span class="sxs-lookup"><span data-stu-id="c9c99-145">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="c9c99-146">ويمكنك تحديد كمية الحد أدنى وكمية الحد الأقصى التي يجب تطبيق التوجيه عليها، ويمكنك تحديد أن التوجيه يجب أن يكون مخصصًا لوحدة مخزون محددة.</span><span class="sxs-lookup"><span data-stu-id="c9c99-146">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="c9c99-147">على سبيل المثال، إذا كانت وحدة القياس هي البالتات، فالأصناف في البالتات يمكن وضعها في موقع محدد.</span><span class="sxs-lookup"><span data-stu-id="c9c99-147">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="c9c99-148">ويمكنك أيضًا تحديد ما إذا كان يمكن تقسيم الكمية عبر مواقع متعددة.</span><span class="sxs-lookup"><span data-stu-id="c9c99-148">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="c9c99-149">مثل رأس توجيه موقع‬، يحتوي كل بند من بنود توجيهات الموقع على رقم تسلسلي يحدد الترتيب الذي يتم به تقييم البنود.</span><span class="sxs-lookup"><span data-stu-id="c9c99-149">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span>

<span data-ttu-id="c9c99-150">تتضمن توجيهات الموقع مستوى إضافي واحد للتفاصيل: *إجراءات توجيهات الموقع*.</span><span class="sxs-lookup"><span data-stu-id="c9c99-150">Location directives have one additional level of detail: *location directive actions*.</span></span> <span data-ttu-id="c9c99-151">ويمكنك تحديد العديد من إجراءات توجيه الموقع لكل بند.</span><span class="sxs-lookup"><span data-stu-id="c9c99-151">You can define multiple location directive actions for each line.</span></span> <span data-ttu-id="c9c99-152">‏‫مرة أخرى، يتم استخدام رقم تسلسلي لتحديد الترتيب الذي يتم به تقييم الإجراءات.</span><span class="sxs-lookup"><span data-stu-id="c9c99-152">Once again, a sequence number is used to determine the order that the actions are assessed in.</span></span> <span data-ttu-id="c9c99-153">وعلى هذا المستوى، يمكن إعداد استعلام لتحديد كيفية العثور على أفضل موقع في المستودع.‬</span><span class="sxs-lookup"><span data-stu-id="c9c99-153">On this level, you can set up a query to define how to find the best location in the warehouse.</span></span> <span data-ttu-id="c9c99-154">ويمكنك أيضًا استخدام إعدادات **الاستراتيجية**المحددة مسبقًا للعثور على موقع أمثل.</span><span class="sxs-lookup"><span data-stu-id="c9c99-154">You can also use predefined **Strategy** settings to find an optimal location.</span></span>
