# <a name="hire-existing-employees-through-recruitment"></a><span data-ttu-id="e36ad-103">تعيين موظفين موجودين من خلال التوظيف</span><span class="sxs-lookup"><span data-stu-id="e36ad-103">Hire existing employees through recruitment</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="e36ad-104">في بعض الأحيان، يمكن شغل مناصب مفتوحة بواسطة مرشحين عاملين في مؤسستك.</span><span class="sxs-lookup"><span data-stu-id="e36ad-104">Sometimes open positions can be filled by candidates who are already employees in your organization.</span></span> <span data-ttu-id="e36ad-105">يوضح هذا الإجراء الخطوات لتوظيف موظف موجود من خلال عملية التعيين.</span><span class="sxs-lookup"><span data-stu-id="e36ad-105">This procedure walks through the steps of hiring an existing employee through the recruiting process.</span></span> <span data-ttu-id="e36ad-106">في هذا الإجراء، تم إعداد مشروع تعيين، وقدم موظف موجود استمارة تقديم لمشروع التعيين.</span><span class="sxs-lookup"><span data-stu-id="e36ad-106">In this procedure, a recruitment project has already been set up, and an existing employee has already submitted an application for the recruitment project.</span></span> <span data-ttu-id="e36ad-107">شركة بيانات العرض التوضيحي التي تم استخدامها لإنشاء هذا الإجراء هي USMF.</span><span class="sxs-lookup"><span data-stu-id="e36ad-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="e36ad-108">انتقل إلى الموارد البشرية > التوظيف‬ > استمارات التقديم‬ > استمارات التقديم‬‬.</span><span class="sxs-lookup"><span data-stu-id="e36ad-108">Go to Human resources > Recruitment > Applications > Applications.</span></span>
2. <span data-ttu-id="e36ad-109">في القائمة، ابحث عن استمارة تقديم للموظف الذي ترغب في توظيفه.</span><span class="sxs-lookup"><span data-stu-id="e36ad-109">In the list, find application for the employee that you would like to hire.</span></span> <span data-ttu-id="e36ad-110">على سبيل المثال: 00002 جون إموري</span><span class="sxs-lookup"><span data-stu-id="e36ad-110">Example:  00002  John Emory</span></span>
3. <span data-ttu-id="e36ad-111">انقر فوق "حالة استمارة التقديم‬".</span><span class="sxs-lookup"><span data-stu-id="e36ad-111">Click Application status.</span></span>
    * <span data-ttu-id="e36ad-112">تشير حالة استمارة التقديم إلى موقع استمارة التقديم في عملية التعيين.</span><span class="sxs-lookup"><span data-stu-id="e36ad-112">The application status indicates where an application is at in the recruitment process.</span></span>  <span data-ttu-id="e36ad-113">كل هذه الخطوات اختيارية.</span><span class="sxs-lookup"><span data-stu-id="e36ad-113">Each of these steps is optional.</span></span> <span data-ttu-id="e36ad-114">بشكل عام، قد تقوم استمارة التقديم بنقل الحالات بالترتيب التالي: مستلمة ومؤكدة وتم إجراء المقابلة.</span><span class="sxs-lookup"><span data-stu-id="e36ad-114">Typically, an application would move the statuses in the following order:  received, confirmed, and interviewed.</span></span> <span data-ttu-id="e36ad-115">بعد عملية إجراء المقابلات، قد يتم اتخاذ قرار التوظيف.</span><span class="sxs-lookup"><span data-stu-id="e36ad-115">After the interview process, a hiring decision would be made.</span></span>  
4. <span data-ttu-id="e36ad-116">انقر فوق "تغيير المنصب".</span><span class="sxs-lookup"><span data-stu-id="e36ad-116">Click Change position.</span></span>
5. <span data-ttu-id="e36ad-117">حدد المنصب الذي تريد توظيف الموظف فيه.</span><span class="sxs-lookup"><span data-stu-id="e36ad-117">Select the position that you are hiring the employee into.</span></span>
6. <span data-ttu-id="e36ad-118">في الحقل "تاريخ بدء التعيين الجديد"، أدخل تاريخ بدء عمل الموظف في المنصب الجديد.</span><span class="sxs-lookup"><span data-stu-id="e36ad-118">In the New Assignment Start Date field, enter the date that the employee will begin working in the new position.</span></span>  
7. <span data-ttu-id="e36ad-119">في الحقل "تاريخ انتهاء التعيين"، أدخل تاريخ توقف الموظف عن العمل في المنصب الحالي.</span><span class="sxs-lookup"><span data-stu-id="e36ad-119">In the Assignment end date, enter the date that the employee will stop working in their current position.</span></span>
    * <span data-ttu-id="e36ad-120">قد يتداخل تاريخ بدء المنصب الجديد مع تاريخ الانتهاء المنصب القديم.</span><span class="sxs-lookup"><span data-stu-id="e36ad-120">The starting date for the new position and the ending date of the old position may overlap.</span></span> <span data-ttu-id="e36ad-121">قد يحدث هذا الأمر عندما يقوم أحد الأشخاص بأداء واجباته في المنصبين خلال فترة انتقالية.</span><span class="sxs-lookup"><span data-stu-id="e36ad-121">This can happen when a person is performing duties for both positions during a transition period.</span></span>  
8. <span data-ttu-id="e36ad-122">بشكل اختياري، يمكنك تحديد كود سبب.</span><span class="sxs-lookup"><span data-stu-id="e36ad-122">Optionally, you can select a reason code.</span></span> <span data-ttu-id="e36ad-123">على سبيل المثال: إعادة التنظيم</span><span class="sxs-lookup"><span data-stu-id="e36ad-123">Example: Reorganization</span></span>
9. <span data-ttu-id="e36ad-124">انقر فوق "تغيير المنصب".</span><span class="sxs-lookup"><span data-stu-id="e36ad-124">Click Change position.</span></span>
    * <span data-ttu-id="e36ad-125">يمكنك أيضًا تغيير التعويض في هذا الوقت.</span><span class="sxs-lookup"><span data-stu-id="e36ad-125">You may also change compensation at this time.</span></span> <span data-ttu-id="e36ad-126">إذا لم تقم بتعيين التعويض في هذا الوقت، فيمكنك تغييره بالانتقال إلى نموذج العامل وتحديد علامة التبويب "التعويض" واختيار "خطة ثابتة‬".</span><span class="sxs-lookup"><span data-stu-id="e36ad-126">If you do not assign compensation at this time, you can change it by going to the worker form, selecting the Compensation tab, and choosing 'Fixed Plan'.</span></span> <span data-ttu-id="e36ad-127">بعد تحديد "تغيير المنصب"، سيتم تحديث حالة استمارة التقديم إلى "موظف".</span><span class="sxs-lookup"><span data-stu-id="e36ad-127">After you select 'Change position', the status on the application will be updated to 'Employed'.</span></span>
