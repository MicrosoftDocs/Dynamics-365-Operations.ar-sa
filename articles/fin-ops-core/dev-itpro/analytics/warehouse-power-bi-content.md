<span data-ttu-id="b1f95-110">باستطاعة مدراء المستودعات استخدام محتوى **أداء المستودع** في Power BI لقياس النواحي الثلاث التالية:</span><span class="sxs-lookup"><span data-stu-id="b1f95-110">Warehouse managers can use the **Warehouse performance** Power BI content to measure the following three areas:</span></span>
