## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="af8d6-108">إضافة حقول إلى ورقة عمل في مصنف Excel</span><span class="sxs-lookup"><span data-stu-id="af8d6-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="af8d6-109">لإضافة حقول إلى مصنف Excel بحيث يمكنك تحرير حركات البيع بالتجزئة، اتبع هذه الخطوات.</span><span class="sxs-lookup"><span data-stu-id="af8d6-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="af8d6-110">قم بتنزيل ملف Excel وافتحه من صفحة **كشوف الحسابات** ، أو صفحة **حركات البيع بالتجزئة** ، أو مربع الإطار المتجانب **إخفاقات التحقق من صحة الحركات** في مساحة العمل **ماليات المتجر**.</span><span class="sxs-lookup"><span data-stu-id="af8d6-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="af8d6-111">حدد **تصميم**.</span><span class="sxs-lookup"><span data-stu-id="af8d6-111">Select **Design**.</span></span>
1. <span data-ttu-id="af8d6-112">حدد رمز القلم للجدول المطلوب، ثم في قائمة الحقول المتاحة، ابحث عن الحقل الذي تريد إضافته وحدده.</span><span class="sxs-lookup"><span data-stu-id="af8d6-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
1. <span data-ttu-id="af8d6-113">حدد **إضافة**، ثم حدد **تحديث**.</span><span class="sxs-lookup"><span data-stu-id="af8d6-113">Select **Add**, and then select **Update**.</span></span> <span data-ttu-id="af8d6-114">يمكنك إعادة ترتيب الحقول.</span><span class="sxs-lookup"><span data-stu-id="af8d6-114">You can reorder fields.</span></span>
1. <span data-ttu-id="af8d6-115">بعد اكتمال التحديث، حدد **‏‫تحديث** لجلب البيانات للعمود الجديد.</span><span class="sxs-lookup"><span data-stu-id="af8d6-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>

<span data-ttu-id="af8d6-116">يتعين أن يكون الحقل الجديد والبيانات الخاصة به متاحة الآن للتحرير في Excel.</span><span class="sxs-lookup"><span data-stu-id="af8d6-116">The new field and data for it should now be available for editing in Excel.</span></span>
