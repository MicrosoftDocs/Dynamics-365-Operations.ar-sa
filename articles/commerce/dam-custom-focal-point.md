# <a name="customize-image-focal-points"></a><span data-ttu-id="fe92c-103">تخصيص نقاط تركيز الصورة</span><span class="sxs-lookup"><span data-stu-id="fe92c-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="fe92c-104">يصف هذا الموضوع كيفية تخصيص نقاط تركيز الصورة في منشئ موقع Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="fe92c-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="fe92c-105">عند تحميل صورة إلى مكتبه الوسائط الخاصة بمنشئ موقع Commerce، يحاول النظام تحديد نقطة تركيز الصورة.</span><span class="sxs-lookup"><span data-stu-id="fe92c-105">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="fe92c-106">على سبيل المثال، عند وجود شخص في الصورة، سيقوم النظام بتعيين نقطة التركيز إلى وجه الشخص بشكل افتراضي.</span><span class="sxs-lookup"><span data-stu-id="fe92c-106">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="fe92c-107">في معظم الحالات، تعمل نقطة التركيز بشكل جيد لجميع مناف العرض، ولكنك قد ترغب في بعض الأحيان بضبط نقطة التركيز للتأكيد من أن جزءًا معينًا من الصورة مرئي دائمًا.</span><span class="sxs-lookup"><span data-stu-id="fe92c-107">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="fe92c-108">تحديد نقطة تركيز مخصصة لصورة</span><span class="sxs-lookup"><span data-stu-id="fe92c-108">Define a custom focal point for an image</span></span>

<span data-ttu-id="fe92c-109">لتحديد نقطة تركيز مخصصة لصورة، اتبع الخطوات التالية.</span><span class="sxs-lookup"><span data-stu-id="fe92c-109">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="fe92c-110">في جزء التنقل الأيمن في منشئ موقع Commerce، حدد **مكتبة الوسائط**.</span><span class="sxs-lookup"><span data-stu-id="fe92c-110">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="fe92c-111">في النافذة الرئيسية،، حدد الصورة التي تريد تعديلها.</span><span class="sxs-lookup"><span data-stu-id="fe92c-111">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="fe92c-112">في شريط الأوامر، حدد **تحرير**.</span><span class="sxs-lookup"><span data-stu-id="fe92c-112">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="fe92c-113">حدد الصورة التي تريد إدخالها في **وضع التحرير**.</span><span class="sxs-lookup"><span data-stu-id="fe92c-113">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="fe92c-114">ضمن **وضع التحرير**، حدد **تغيير نقطة التركيز**.</span><span class="sxs-lookup"><span data-stu-id="fe92c-114">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="fe92c-115">يظهر عنصر تحكم نقطة تركيز فوق الصورة.</span><span class="sxs-lookup"><span data-stu-id="fe92c-115">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="fe92c-116">حدد عنصر تحكم نقطة التركيز لتحريكه فوق نقطة التركيز التي تريدها.</span><span class="sxs-lookup"><span data-stu-id="fe92c-116">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="fe92c-117">عند الانتهاء، حدد **حفظ** على شريط الأوامر، ثم حدد **إنهاء التحرير**.</span><span class="sxs-lookup"><span data-stu-id="fe92c-117">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
