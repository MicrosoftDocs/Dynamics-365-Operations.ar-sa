1. <span data-ttu-id="5c36d-113">ضمن **وضع التحرير**، حدد **تحرير طريقة العرض حسب الوحدة النمطية**.</span><span class="sxs-lookup"><span data-stu-id="5c36d-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>

1. <span data-ttu-id="5c36d-116">من القائمة المنسدلة **الموضع**، حدد موضع الصورة.</span><span class="sxs-lookup"><span data-stu-id="5c36d-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="5c36d-117">من القائمة المنسدلة **منفذ العرض**، حدد حجم منفذ العرض.</span><span class="sxs-lookup"><span data-stu-id="5c36d-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="5c36d-118">تتراكب المنطقة التي تمثل منطقة الاقتصاص على الصورة.</span><span class="sxs-lookup"><span data-stu-id="5c36d-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="5c36d-119">يمكنك تحريك منطقة الاقتصاص وتغيير حجمها حسب الحاجة.</span><span class="sxs-lookup"><span data-stu-id="5c36d-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="5c36d-120">سيتم الاحتفاظ بنسبة العرض إلى الارتفاع بشكل تلقائي.</span><span class="sxs-lookup"><span data-stu-id="5c36d-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="5c36d-121">عند الانتهاء، حدد **حفظ** على شريط الأوامر، ثم حدد **إنهاء التحرير**.</span><span class="sxs-lookup"><span data-stu-id="5c36d-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="5c36d-122">بعد إتمام الاقتصاص المخصص، ستدخل تعديلات الصورة حيز التنفيذ على الفور.</span><span class="sxs-lookup"><span data-stu-id="5c36d-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
