[!include [banner](includes/banner.md)]

<span data-ttu-id="34e85-104">‏‫يوضح هذا الموضوع كيفية إضافة أيقونة مفضلة إلى الموقع الخاص بك.‬</span><span class="sxs-lookup"><span data-stu-id="34e85-104">This topic explains how to add a favicon to your site.</span></span>

<span data-ttu-id="34e85-105">وأيقونة المفضلة هي ملف رسومي صغير يتم عرضة على علامة تبويب مستعرض ويب، في شريط العنوان، وفي محفوظات الاستعراض، وفي الإشارات المرجعية أو المفضلة، من بين أماكن أخرى.</span><span class="sxs-lookup"><span data-stu-id="34e85-105">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="34e85-106">نوصي أن تقوم بإضافة أيقونة المفضلة إلى موقعك، لأنها تعرض وتعزز علامتك التجارية، وتساعد في تمييز موقعك عن المواقع الأخرى التي يزورها عملائك.</span><span class="sxs-lookup"><span data-stu-id="34e85-106">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="34e85-107">على الرغم من أنه يُمكنك إضافة عدة أيقونات للمفضلة بأحجام مختلفة وأنواع ملفات إلى موقعك، ويوضح هذا الموضوع كيفية إضافة أيقونة مفضلة واحدة.</span><span class="sxs-lookup"><span data-stu-id="34e85-107">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="34e85-108">ولكن، يتم استخدام نفس العملية والموقع لإضافة أكثر من أيقونة المفضلة.</span><span class="sxs-lookup"><span data-stu-id="34e85-108">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="34e85-109">تحميل أيقونة المفضلة إلى مجموعة أصول الموقع الخاص بك</span><span class="sxs-lookup"><span data-stu-id="34e85-109">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="34e85-110">لتحميل أيقونة المفضلة إلى مجموعة أصول الموقع الخاص بك، اتبع الخطوات التالية.</span><span class="sxs-lookup"><span data-stu-id="34e85-110">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="34e85-112">في شريط الأوامر، حدد **تحميل \> تحميل عناصر الوسائط**.</span><span class="sxs-lookup"><span data-stu-id="34e85-112">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="34e85-113">في نافذة "مستكشف الملفات"، استعرض بحثا عن ملف صورة أيقونة المفضلة‬ الذي تريد تحميله، وحدده، ثم حدد **فتح**.</span><span class="sxs-lookup"><span data-stu-id="34e85-113">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>

1. <span data-ttu-id="34e85-115">إذا أردت نشر الصورة مباشرةً بعد التحميل، فحدد خانه الاختيار **نشر عناصر الوسائط بعد التحميل**.</span><span class="sxs-lookup"><span data-stu-id="34e85-115">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="34e85-116">إذا لم تقم بتحديد خانة الاختيار **نشر عناصر الوسائط بعد التحميل‬‏‫**، يجب عليك العودة إلى صفحة **عناصر الوسائط**، ونشر أيقونة المفضلة يدويًا في وقت لاحق.</span><span class="sxs-lookup"><span data-stu-id="34e85-116">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="34e85-117">حدد **موافق**.</span><span class="sxs-lookup"><span data-stu-id="34e85-117">Select **OK**.</span></span>
1. <span data-ttu-id="34e85-118">في جزء الخصائص الموجود على اليمين، انسخ عنوان URL العمومي لأيقونة المفضلة.</span><span class="sxs-lookup"><span data-stu-id="34e85-118">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="34e85-119">ستستخدم عنوان URL هذا لاحقا.</span><span class="sxs-lookup"><span data-stu-id="34e85-119">You will use this URL later.</span></span>

1. <span data-ttu-id="34e85-134">في شجرة التدرج الهرمي للقالب، حدد زر علامة القطع (**...**) إلى يمين حاوية **رأس HTML**، ثم حدد **إضافة جزء**.</span><span class="sxs-lookup"><span data-stu-id="34e85-134">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add fragment**.</span></span>
