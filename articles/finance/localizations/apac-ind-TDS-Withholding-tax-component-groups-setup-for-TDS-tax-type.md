# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="7e5f3-103">إعداد مجموعات مكون ضريبة الخصم لنوع ضريبة TDS</span><span class="sxs-lookup"><span data-stu-id="7e5f3-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="7e5f3-104">يوضح هذا الموضوع كيفية إعداد مجموعات مكون ضريبة الخصم، مثل **الإيجار** و **المقاول**، لنوع الضريبة المخصومة في المصدر (TDS).</span><span class="sxs-lookup"><span data-stu-id="7e5f3-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="7e5f3-105">انتقل إلى **الضريبة \> الإعداد \> ضريبة الخصم \> مجموعة مكون ضريبة الخصم**.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="7e5f3-106">[![صفحة مجموعات مكون ضرائب الخصم](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="7e5f3-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="7e5f3-107">في الحقل **نوع الضريبة**، حدد **TDS** لإعداد مجموعات مكون ضريبة الخصم لنوع الضريبة TDS.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="7e5f3-108">في جزء الإجراءات، حدد **جديد** لإنشاء بند.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="7e5f3-109">في الحقل **مجموعة مكون ضريبة الخصم**، أدخل اسمًا لمجموعة مكون TDS.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="7e5f3-110">في الحقل **الوصف**، أدخل وصفًا.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="7e5f3-111">في علامات التبويب السريعة **عام**، في الحقل **الحالة**، حدد الحالة المتضمنة لمجموعة مكون TDS.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="7e5f3-112">تكون الخيارات **مقيم** و **غير مقيم**.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="7e5f3-113">في الحقل **كود القسم**، أدخل كود القسم الذي يتم تطبيقه على مجموعة مكون TDS.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="7e5f3-114">قم بإغلاق الصفحة.</span><span class="sxs-lookup"><span data-stu-id="7e5f3-114">Close the page.</span></span>
