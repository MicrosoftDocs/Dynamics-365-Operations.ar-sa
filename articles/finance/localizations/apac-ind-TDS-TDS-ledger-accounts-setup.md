# <a name="set-up-tds-ledger-accounts"></a><span data-ttu-id="38223-103">إعداد حسابات دفتر الأستاذ الخاص بـ TDS</span><span class="sxs-lookup"><span data-stu-id="38223-103">Set up TDS ledger accounts</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="38223-104">يوضح هذا الموضوع كيفية إعداد حسابات دفتر الأستاذ لميزة الضريبة المخصومة في المصدر (TDS).</span><span class="sxs-lookup"><span data-stu-id="38223-104">This topic explains how to set up ledger accounts for the Tax Deducted at Source (TDS) feature.</span></span> <span data-ttu-id="38223-105">استخدم الصفحة **دليل الحسابات** لإعداد حسابات دفتر الأستاذ من أجل TDS.</span><span class="sxs-lookup"><span data-stu-id="38223-105">You use the **Chart of accounts** page to set up ledger accounts for TDS.</span></span>

1. <span data-ttu-id="38223-106">انتقل **دفتر الأستاذ العام \> دليل الحسابات \> الحسابات \> الحسابات الرئيسية**.</span><span class="sxs-lookup"><span data-stu-id="38223-106">Go to **General ledger \> Chart of accounts \> Accounts \> Main accounts**.</span></span>
2. <span data-ttu-id="38223-107">في علامة التبويب **نظرة عامة**، حدد **Alt + N** لإنشاء حساب دفتر الأستاذ الخاص بـ TDS، وأدخل التفاصيل المطلوبة.</span><span class="sxs-lookup"><span data-stu-id="38223-107">On the **Overview** tab, select **Alt+N** to create a TDS ledger account, and enter the required details.</span></span>
3. <span data-ttu-id="38223-108">في علامة التبويب **إعداد**، في الحقل **نوع الترحيل**، حدد **ضريبة الخصم**.</span><span class="sxs-lookup"><span data-stu-id="38223-108">On the **Setup** tab, in the **Posting type** field, select **Withholding tax**.</span></span>     

    > [!NOTE]
    > <span data-ttu-id="38223-109">يمكن تحديد حسابات دفتر الأستاذ التي تحتوي على نوع ترحيل **ضريبة الخصم** فقط كحسابات مستحقة يتم استخدامها لترحيل مبلغ TDS مستحق القبض لكود ضريبة TDS.</span><span class="sxs-lookup"><span data-stu-id="38223-109">Ledger accounts that have a posting type of **Withholding tax** can be defined only as receivable accounts that are used to post the TDS receivable amount for a TDS tax code.</span></span>

4. <span data-ttu-id="38223-110">قم بإغلاق الصفحة.</span><span class="sxs-lookup"><span data-stu-id="38223-110">Close the page.</span></span>
