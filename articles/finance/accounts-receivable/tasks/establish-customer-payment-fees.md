# <a name="establish-customer-payment-fees"></a><span data-ttu-id="521db-103">‏‫وضع رسوم دفع العميل‬</span><span class="sxs-lookup"><span data-stu-id="521db-103">Establish customer payment fees</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="521db-104">إنشاء رسوم الدفع لمدفوعات العميل.</span><span class="sxs-lookup"><span data-stu-id="521db-104">Create payment fees for customer payments.</span></span>

<span data-ttu-id="521db-105">تستخدم هذه المهمة شركة بيانات العرض التوضيحي USMF.</span><span class="sxs-lookup"><span data-stu-id="521db-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="521db-106">في **جزء التنقل**، انتقل إلى **الوحدات النمطية‬ > حسابات مدينة‬ > إعداد المدفوعات‬ > رسوم الدفع‬**.</span><span class="sxs-lookup"><span data-stu-id="521db-106">In the **Navigation pane**, go to **Modules > Accounts receivable > Payments setup > Payment fee**.</span></span>
2. <span data-ttu-id="521db-107">انقر فوق **جديد**.</span><span class="sxs-lookup"><span data-stu-id="521db-107">Click **New**.</span></span>
3. <span data-ttu-id="521db-108">في الحقل **معرف الرسوم**، أدخل معرف رسوم.</span><span class="sxs-lookup"><span data-stu-id="521db-108">In the **Fee ID** field, enter a Fee ID.</span></span> <span data-ttu-id="521db-109">يظهر "معرف الرسوم" على دفاتر يومية المدفوعات، لذلك اجعله وصفيًا لفهم ما هي الرسوم التي يتم فرضها.</span><span class="sxs-lookup"><span data-stu-id="521db-109">The Fee ID displays on payment journals, so make it descriptive to understand what fee is being assessed.</span></span>

5. <span data-ttu-id="521db-111">أدخل وصفًا للرسوم في حقل **وصف الرسوم**.</span><span class="sxs-lookup"><span data-stu-id="521db-111">In the **Fee description** field, enter a description for the fee.</span></span>
6. <span data-ttu-id="521db-112">في حقل **التكلفة**، حدد ما إذا كان سيتم تحميل الرسوم لحساب العميل أو حساب دفتر أستاذ.</span><span class="sxs-lookup"><span data-stu-id="521db-112">In the **Charge** field, select whether the fee will be charged to the Customer or a Ledger account.</span></span> <span data-ttu-id="521db-113">إذا كانت الرسوم ستُفرض على العميل، فحدد "العميل".</span><span class="sxs-lookup"><span data-stu-id="521db-113">If the customer is assessed the fee, select Customer.</span></span> <span data-ttu-id="521db-114">إذا كانت الرسوم ستُفرض على مؤسستك كمصروفات، فحدد "دفتر الأستاذ‬".</span><span class="sxs-lookup"><span data-stu-id="521db-114">If the fee will be assess to your organization as an expense, select Ledger.</span></span> <span data-ttu-id="521db-115">لهذه المهمة، حدد "العميل".</span><span class="sxs-lookup"><span data-stu-id="521db-115">For this task, select Customer.</span></span>  
7. <span data-ttu-id="521db-116">في الحقل **نوع دفتر اليومية**، حدد نوع دفتر اليومية الذي يمكنه استخدام رسوم الدفع هذه.</span><span class="sxs-lookup"><span data-stu-id="521db-116">In the **Journal type** field, select the type of journal that can use this payment fee.</span></span> <span data-ttu-id="521db-117">إذا كانت هذه الرسوم ستُستخدم لمدفوعات العميل، فسيكون نوع دفتر اليومية على الأرجح دفع العميل‬.</span><span class="sxs-lookup"><span data-stu-id="521db-117">If these fees are used for customer payments, the journal type will likely be Customer payment.</span></span>  
8. <span data-ttu-id="521db-118">انقر فوق **حفظ**.</span><span class="sxs-lookup"><span data-stu-id="521db-118">Click **Save**.</span></span>
9. <span data-ttu-id="521db-119">انقر فوق **إعداد رسوم الدفع**.</span><span class="sxs-lookup"><span data-stu-id="521db-119">Click **Payment fee setup**.</span></span> <span data-ttu-id="521db-120">يتم استخدام إعداد رسوم الدفع لتعريف المعايير عند فرض رسوم الدفع.</span><span class="sxs-lookup"><span data-stu-id="521db-120">The Payment fee setup is used to define the criteria for when the payment fee will be assessed.</span></span>  <span data-ttu-id="521db-121">على سبيل المثال، يمكنك تحديد أنه سيتم حساب الرسوم إذا كان الحساب البنكي USMF OPER، وطريقة الدفع هي الشيك.</span><span class="sxs-lookup"><span data-stu-id="521db-121">For example, you can define that the fee will be calculated if the bank account is USMF OPER, and the method of payment is check.</span></span>

11. <span data-ttu-id="521db-126">في الحقل **علاقة البنك**، حدد مجموعة بنكية أو حساب بنكي.</span><span class="sxs-lookup"><span data-stu-id="521db-126">In the **Bank relation** field, select either a bank group or a bank account.</span></span> <span data-ttu-id="521db-127">إذا حددت "الجدول"، فسيعرض البحث الحسابات البنكية.</span><span class="sxs-lookup"><span data-stu-id="521db-127">If you selected Table, the lookup will display bank accounts.</span></span> <span data-ttu-id="521db-128">أما إذا حددت "المجموعة"، فسيعرض البحث المجموعات البنكية.</span><span class="sxs-lookup"><span data-stu-id="521db-128">If you selected Group, the lookup will display bank groups.</span></span>  
12. <span data-ttu-id="521db-129">في القائمة، انقر فوق الارتباط في الصف المحدد.</span><span class="sxs-lookup"><span data-stu-id="521db-129">In the list, click the link in the selected row.</span></span>
13. <span data-ttu-id="521db-130">في الحقل **طريقة الدفع**، حدد طريقه الدفع التي سيتم تقييم هذه الرسوم لها.</span><span class="sxs-lookup"><span data-stu-id="521db-130">In the **Method of payment** field, select the Method of payment for which this fee will be assessed.</span></span> <span data-ttu-id="521db-131">على سبيل المثال، يمكنك فرض رسوم على العملاء إذا قاموا بإرسال الدفعات كشيك، بدلاً من دفعة إلكترونية.</span><span class="sxs-lookup"><span data-stu-id="521db-131">For example, you may assess a fee to your customers if they send payments as a check, rather than as an electronic payment.</span></span>  
14. <span data-ttu-id="521db-132">في القائمة، قم بالبحث عن السجل المطلوب وحدده.</span><span class="sxs-lookup"><span data-stu-id="521db-132">In the list, find and select the desired record.</span></span>
15. <span data-ttu-id="521db-133">أدخل عملة دفع، إذا كان ذلك ملائمًا، في حقل **عملة الدفع**.</span><span class="sxs-lookup"><span data-stu-id="521db-133">If relevant, in the **Payment currency** field, enter a payment currency.</span></span> <span data-ttu-id="521db-134">يتم استخدام عملة الدفع كمعيار إضافي لما إذا كان سيتم فرض الرسوم.</span><span class="sxs-lookup"><span data-stu-id="521db-134">The payment currency is used as an additional criteria for whether the fee will be assessed.</span></span>  <span data-ttu-id="521db-135">على سبيل المثال، قد يفرض البنك رسمًا إضافيًا للدفعات المستلمة بعملة الدولار الأمريكي، إذ يقوم عادةً بتنفيذ المعاملات باليورو فقط.</span><span class="sxs-lookup"><span data-stu-id="521db-135">For example, your bank may charge an extra fee for payments received in USD currency, since they normally only transact in EUR currency.</span></span>  
16. <span data-ttu-id="521db-136">حدد ما إذا كانت الرسوم ستكون عبارة عن نسبة أو مبلغ أو فاصل زمني.</span><span class="sxs-lookup"><span data-stu-id="521db-136">Select whether the fee will be a percent, amount or interval.</span></span>
17. <span data-ttu-id="521db-137">في الحقل **النسبة المئوية/المبلغ**، أدخل نسبة مئوية أو مبلغًا للرسوم.</span><span class="sxs-lookup"><span data-stu-id="521db-137">In the **Percentage/Amount field**, enter either percentage or amount of the fee.</span></span> <span data-ttu-id="521db-138">إذا كان حقل "النسبة المئوية/المبلغ" معينًا إلى "النسبة‬", فستكون عندئذٍ القيمة المدخلة هنا نسبة مئوية.</span><span class="sxs-lookup"><span data-stu-id="521db-138">If the Percentage/Amount field is Percent, then the value enter here will be a percentage.</span></span> <span data-ttu-id="521db-139">وإذا كان حقل "النسبة المئوية/المبلغ" معينًا إلى "المبلغ", فستكون عندئذٍ القيمة التي تدخلها هنا مبلغًا.</span><span class="sxs-lookup"><span data-stu-id="521db-139">If the Percentage/Amount field is Amount, then the value you enter here will be an amount.</span></span> <span data-ttu-id="521db-140">أما إذا كان حقل "النسبة المئوية/المبلغ" معينًا إلى "الفاصل الزمني‬", فاستخدم علامة التبويب "الفاصل الزمني‬" لتحديد المستويات.</span><span class="sxs-lookup"><span data-stu-id="521db-140">If the Percentage/Amount field is Interval, use the Interval tab to define the tiers.</span></span>  
18. <span data-ttu-id="521db-141">في حقل **عملة الرسوم**، حدد عملة الرسوم.</span><span class="sxs-lookup"><span data-stu-id="521db-141">In the **Fee currency** field, select the currency of the fee.</span></span> <span data-ttu-id="521db-142">هذه هي العملة التي سيتم استخدامها لإنشاء الرسوم.</span><span class="sxs-lookup"><span data-stu-id="521db-142">This is the currency in which the fee will be created.</span></span>  
19. <span data-ttu-id="521db-143">انقر فوق **حفظ**.</span><span class="sxs-lookup"><span data-stu-id="521db-143">Click **Save**.</span></span>
