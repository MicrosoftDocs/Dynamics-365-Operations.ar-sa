[!include [banner](../../includes/banner.md)]

<span data-ttu-id="30dec-104">يوضح هذا الإجراء كيفية إنشاء اسم مستعار لحساب يوفر اختصارًا لإدخال رقم حساب.</span><span class="sxs-lookup"><span data-stu-id="30dec-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="30dec-105">يستخدم هذا الإجراء شركة بيانات العرض التوضيحي USMF.</span><span class="sxs-lookup"><span data-stu-id="30dec-105">This procedure uses demo data company USMF.</span></span>

1. <span data-ttu-id="30dec-106">انتقل إلى دفتر الأستاذ العام > دليل الحسابات > الحسابات > الاسم المستعار لحساب دفتر الأستاذ.</span><span class="sxs-lookup"><span data-stu-id="30dec-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>

3. <span data-ttu-id="30dec-108">في الحقل "الاسم المستعار لحساب دفتر الأستاذ"، اكتب قيمة.</span><span class="sxs-lookup"><span data-stu-id="30dec-108">In the Ledger account alias field, type a value.</span></span>

8. <span data-ttu-id="30dec-113">في الحقل "تعريف الاسم المستعار لحساب دفتر الأستاذ"، حدد الحساب والأبعاد.</span><span class="sxs-lookup"><span data-stu-id="30dec-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>

9. <span data-ttu-id="30dec-115">في الحقل "التركيز الأولي"، حدد البُعد الذي سيتم التركيز عليه عند استخدام الاسم المستعار.</span><span class="sxs-lookup"><span data-stu-id="30dec-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="30dec-116">بعد كتابة الاختصار وملء الحساب والأبعاد، سيكون مجال التركيز الأولى حيث يتحرك المؤشر أو حيث سيتحرك التركيز.</span><span class="sxs-lookup"><span data-stu-id="30dec-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
