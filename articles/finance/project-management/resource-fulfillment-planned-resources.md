[!include [banner](../includes/banner.md)]

<span data-ttu-id="56b0e-104">بإمكان مدير المشروع أن يخطط لأدوار الموارد المطلوبة للمشروع.</span><span class="sxs-lookup"><span data-stu-id="56b0e-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="56b0e-105">سوف يرى مدير الموارد هذه الموارد المخططة كطلبات في صفحة **تنفيذ المورد‬** ويمكنه تعيين الموارد الفعلية.</span><span class="sxs-lookup"><span data-stu-id="56b0e-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="56b0e-106">في صفحة **كافة المشاريع**، حدد المشروع **المرحلة 2 من ترقية XYZ**.</span><span class="sxs-lookup"><span data-stu-id="56b0e-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="56b0e-107">حدد **المشروع**، ثم حدد **تحرير**.</span><span class="sxs-lookup"><span data-stu-id="56b0e-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="56b0e-108">في علامة التبويب **فريق المشروع وجدولته**، حدد **إضافة**.</span><span class="sxs-lookup"><span data-stu-id="56b0e-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="56b0e-109">في مربع الحوار **إضافة أدوار**، حدد الدور **مطور برامج**.</span><span class="sxs-lookup"><span data-stu-id="56b0e-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="56b0e-110">حدد **إنشاء** ثم قم بإغلاق صفحة المشروع.</span><span class="sxs-lookup"><span data-stu-id="56b0e-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="56b0e-111">في صفحة **تنفيذ المورد**، حدد **مطور برامج 1** للمشروع **المرحلة 2 من مشروع ترقية XYZ**.</span><span class="sxs-lookup"><span data-stu-id="56b0e-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="56b0e-112">حدد عاملاً، ثم حدد **تعيين**.</span><span class="sxs-lookup"><span data-stu-id="56b0e-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="56b0e-113">تأكد من إزالة سطر **مطور برامج 1** للمشروع **المرحلة 2 من مشروع ترقية XYZ**.</span><span class="sxs-lookup"><span data-stu-id="56b0e-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="56b0e-114">على علامة التبويب **فريق المشروع وجدولته**، للمشروع **المرحلة 2 من ترقية XYZ**، تأكد من إضافة العامل الذي قمت بتحديده في الخطوة السابقة على أنه **مطور برامج**.</span><span class="sxs-lookup"><span data-stu-id="56b0e-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
