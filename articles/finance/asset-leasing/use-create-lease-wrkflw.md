<span data-ttu-id="d5faa-109">قبل استخدام هذا الإجراء، تأكد من إنشاء سير عمل للموافقة على الإيجار على الأقل.</span><span class="sxs-lookup"><span data-stu-id="d5faa-109">Before you use this procedure, make sure that at least on lease approval workflow has been created.</span></span> <span data-ttu-id="d5faa-110">في حالة عدم وجود سير عمل، قم بإنشاء واحدًا.</span><span class="sxs-lookup"><span data-stu-id="d5faa-110">If no workflow exists, create one.</span></span> <span data-ttu-id="d5faa-111">للحصول على معلومات حول كيفية إعداد سير العمل، راجع [إعداد عمليات سير عمل معتمدة لعقد الإيجار](set-up-lease-wrkflw.md).</span><span class="sxs-lookup"><span data-stu-id="d5faa-111">For information about how to set up a workflow, see [Set up lease approval workflows](set-up-lease-wrkflw.md).</span></span>
