# <a name="view-an-expense-report-from-general-ledger-or-procurement-and-sourcing"></a><span data-ttu-id="5b564-103">عرض تقرير مصروفات من دفتر الأستاذ العام أو التدبير والتوريد</span><span class="sxs-lookup"><span data-stu-id="5b564-103">View an expense report from General ledger or Procurement and sourcing</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="5b564-104">عندما تقوم بتشغيل استعلام لعرض الحركات في دفتر الأستاذ العام أو التدبير والتوريد، فقد ينتج عن الاستعلام الكثير من المعلومات.</span><span class="sxs-lookup"><span data-stu-id="5b564-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="5b564-105">ولذلك، فقد تحتاج إلى تضييق نطاق البحث.</span><span class="sxs-lookup"><span data-stu-id="5b564-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="5b564-106">للحصول على المعلومات التي تحتاجها يمكنك التنقل لأسفل داخل الحركة للاطلاع على تقرير المصروفات الذي تظهر فيه المعلومات.</span><span class="sxs-lookup"><span data-stu-id="5b564-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="5b564-107">بالنسبة لتقارير المصروفات، يمكن التصفح لأسفل من حركة إلى تقرير أصلي تم تخزينه في إدارة المصروفات.</span><span class="sxs-lookup"><span data-stu-id="5b564-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="5b564-108">بالإضافة إلى إجمالي مبلغ المصروفات، يمكنك عرض الحركات المفصلة، والحركات المستحقة لاسترداد ضريبة القيمة المضافة، والإيصالات المرفقة بالمصروفات.</span><span class="sxs-lookup"><span data-stu-id="5b564-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="5b564-109">للتصفح لأسفل في تقرير المصروفات الذي ظهرت فيه الحركة، حدد الحركة، ثم حدد **المستند الأصلي**.</span><span class="sxs-lookup"><span data-stu-id="5b564-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="5b564-110">عندما تقوم بعرض الحركات الفردية، يمكنك أيضًا عرض أي بيانات إضافية مرتبطة بالحركة.</span><span class="sxs-lookup"><span data-stu-id="5b564-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="5b564-111">على سبيل المثال، لمصروفات فندق مُفصلة، يمكنك عرض قائمة مصروفات مفصلة مثل مصروفات استخدام الإنترنت والمكالمات الهاتفية وخدمة الغرف.</span><span class="sxs-lookup"><span data-stu-id="5b564-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
