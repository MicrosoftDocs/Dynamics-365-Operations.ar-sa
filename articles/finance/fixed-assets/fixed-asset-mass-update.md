# <a name="fixed-asset-mass-update"></a><span data-ttu-id="e0c21-103">تحديث شامل للأصول الثابتة</span><span class="sxs-lookup"><span data-stu-id="e0c21-103">Fixed asset mass update</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="e0c21-104">في حالة استخدام الدفاتر، فإنه يمكن تغيير قواعد الإهلاك لمجموعات أصول تعد جزءًا من نفس الدفتر.</span><span class="sxs-lookup"><span data-stu-id="e0c21-104">If you use books, you can change the depreciation conventions for groups of assets that are part of the same book.</span></span>

<span data-ttu-id="e0c21-105">على سبيل المثال، إذا كنت في الولايات المتحدة ووضعت ما يزيد عن 40 بالمائة من الأصول التي لديك في الخدمة أثناء الربع الأخير من العام، فإنه يجب عليك استخدام قاعدة الإهلاك منتصف الربع.</span><span class="sxs-lookup"><span data-stu-id="e0c21-105">For example, if you are in the United States, and you put more than 40 percent of your assets in service during the fourth quarter of the year, you must use the mid-quarter depreciation convention.</span></span> <span data-ttu-id="e0c21-106">يمكن الاستعانة بعملية تحديث شاملة لتغيير جميع الأصول التي تتطلب قاعدة إهلاك جديدة.</span><span class="sxs-lookup"><span data-stu-id="e0c21-106">You can use the process for a mass update to change all assets that require the new depreciation convention.</span></span> 

<span data-ttu-id="e0c21-107">وعندما تقوم بتحديث قواعد الإهلاك للأصول، يمكنك حذف كافة حركات الإهلاك الموجودة لتلك الأصول.</span><span class="sxs-lookup"><span data-stu-id="e0c21-107">When you update the depreciation convention for assets, you delete all depreciation transactions that exist for those assets.</span></span> <span data-ttu-id="e0c21-108">كما تقوم بحذف جميع الحركات الخاصة بعمليات تسوية الإهلاك، والحركات الخاصة بالإهلاك الإضافي، وحركات الإهلاك غير العادية لتلك الأصول.</span><span class="sxs-lookup"><span data-stu-id="e0c21-108">You also delete all transactions for depreciation adjustments, transactions for bonus depreciation, and transactions for extraordinary depreciation for those assets.</span></span> 

<span data-ttu-id="e0c21-109">لتحديث قاعدة الإهلاك لأصول تم التخلص منها بالفعل، فإنه يجب أولاً حذف حركات التخلص الموجودة.</span><span class="sxs-lookup"><span data-stu-id="e0c21-109">To update the depreciation convention for assets that have already been disposed of, you must first delete the existing disposal transactions.</span></span> <span data-ttu-id="e0c21-110">ويجب عليك أيضًا حذف كافة الحركات التي تم إنشاؤها بسبب عملية التخلص منها.</span><span class="sxs-lookup"><span data-stu-id="e0c21-110">You must also delete all transactions that were generated because of the disposal process.</span></span> 

<span data-ttu-id="e0c21-111">وبعد قيامك بتحديث قواعد الإهلاك للأصول، يمكنك معالجة الإهلاك والإهلاك الاستثنائي لكل أصل من الأصول.</span><span class="sxs-lookup"><span data-stu-id="e0c21-111">After you update the depreciation convention for assets, you can process depreciation and extraordinary depreciation for each asset.</span></span> <span data-ttu-id="e0c21-112">كما يمكنك إجراء تسويات الإهلاك اليدوي، إذا كانت هناك حاجة إلى أي تسويات.</span><span class="sxs-lookup"><span data-stu-id="e0c21-112">You can also make manual depreciation adjustments, if any adjustments are required.</span></span>
