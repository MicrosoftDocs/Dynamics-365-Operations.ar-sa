# <a name="change-the-banner-or-logo"></a><span data-ttu-id="791ae-103">تغيير اللافتة أو الشعار</span><span class="sxs-lookup"><span data-stu-id="791ae-103">Change the banner or logo</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="791ae-104">يسرد الإجراء التالي الخطوات التي يمكن لمسؤولي النظام استخدامها لتحديث صورة الشعار أو اللافتة التي يتم عرضها لكيان قانوني.</span><span class="sxs-lookup"><span data-stu-id="791ae-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="791ae-105">شركة بيانات العرض التوضيحي التي تم استخدامها لإنشاء هذا الإجراء هي USMF.</span><span class="sxs-lookup"><span data-stu-id="791ae-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="791ae-106">انتقل إلى **جزء التنقل > الوحدات > إدارة المؤسسة > المؤسسات > الكيانات القانونية**.</span><span class="sxs-lookup"><span data-stu-id="791ae-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="791ae-107">في القائمة الموجودة على اليسار، حدد الكيان القانوني الذي تريد تحديث اللافتة أو الشعار له.</span><span class="sxs-lookup"><span data-stu-id="791ae-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="791ae-108">إذا كان محددًا بالفعل، فانتقل إلى الخطوة التالية.</span><span class="sxs-lookup"><span data-stu-id="791ae-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="791ae-109">قم بتوسيع علامة تبويب **صورة لوحة المعلومات**.</span><span class="sxs-lookup"><span data-stu-id="791ae-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="791ae-110">حدد **تغيير**.</span><span class="sxs-lookup"><span data-stu-id="791ae-110">Select **Change**.</span></span>
    
    <span data-ttu-id="791ae-111">الدقة المثالية لصورة اللافتة هي 1920 × 281 بكسل.</span><span class="sxs-lookup"><span data-stu-id="791ae-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="791ae-112">العرض المثالي لصورة الشعار هي 350 بكسل.</span><span class="sxs-lookup"><span data-stu-id="791ae-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="791ae-113">حدد **حفظ**.</span><span class="sxs-lookup"><span data-stu-id="791ae-113">Select **Save**.</span></span>
6. <span data-ttu-id="791ae-114">انتقل إلى **جزء التنقل > الوحدات > عام > عام > لوحة المعلومات الافتراضية**.</span><span class="sxs-lookup"><span data-stu-id="791ae-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="791ae-115">يجب أن تشاهد صورة اللافتة أو الشعار الجديد على لوحة المعلومات.</span><span class="sxs-lookup"><span data-stu-id="791ae-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="791ae-116">قد تحتاج إلى تغيير الشركة الخاصة بك، باستخدام منتقي الشركة، إلى الشركة التي قمت بتحميل اللافتة لها.</span><span class="sxs-lookup"><span data-stu-id="791ae-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
