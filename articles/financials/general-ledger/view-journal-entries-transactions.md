## <a name="voucher-transactions"></a><span data-ttu-id="2be02-107">حركات الإيصال</span><span class="sxs-lookup"><span data-stu-id="2be02-107">Voucher transactions</span></span>
<span data-ttu-id="2be02-108">**حركات الإيصال** هي صفحة استعلام يمكنك فيها تحديد من الجداول والحقول المختلفة لتحديد معايير للرصيد أو الحركة التي تبحث عنها.</span><span class="sxs-lookup"><span data-stu-id="2be02-108">**Voucher transactions** is an inquiry page where you can select from various tables and fields to specify criteria for the balance or transaction that you're searching for.</span></span> <span data-ttu-id="2be02-109">على سبيل المثال، يمكنك عرض كافة الحركات لحساب أو تاريخ معين أو كافة الحركات من نوع **التشغيل** والموجودة في طبقة ترحيل معينة.</span><span class="sxs-lookup"><span data-stu-id="2be02-109">For example, you can view all transactions for a specific date or account, or all transactions of the **Operating** type that are in a specific posting layer.</span></span> <span data-ttu-id="2be02-110">وبشكل افتراضي، تعرض الصفحة رقم دفتر اليومية، والإيصال، والتاريخ، والحساب الرئيسي، ولكن يمكنك إضافة جداول وحقول ومعايير إضافية لتضييق نطاق البحث.</span><span class="sxs-lookup"><span data-stu-id="2be02-110">By default, the page shows the journal number, voucher, date, and main account, but you can add additional tables, fields, and criteria to narrow down your search.</span></span>

## <a name="audit-trail"></a><span data-ttu-id="2be02-111">سجل المراجعة</span><span class="sxs-lookup"><span data-stu-id="2be02-111">Audit trail</span></span>
<span data-ttu-id="2be02-112">**سجل المراجعة** عبارة عن صفحة استعلام تعرض أنواع الحركات، والأوصاف، والشخص الذي تم إنشاء الحركات بواسطته، ومتى تم إنشاؤها.</span><span class="sxs-lookup"><span data-stu-id="2be02-112">**Audit trail** is an inquiry page that shows the types of transactions, descriptions, who the transactions were created by, and when they were created.</span></span> <span data-ttu-id="2be02-113">ومن صفحة استعلام **سجل المراجعة**، يمكنك عرض حركات الإيصال.</span><span class="sxs-lookup"><span data-stu-id="2be02-113">From the **Audit trail** inquiry page, you can view the voucher transactions.</span></span>

## <a name="financial-reports"></a><span data-ttu-id="2be02-114">التقارير المالية</span><span class="sxs-lookup"><span data-stu-id="2be02-114">Financial reports</span></span>
<span data-ttu-id="2be02-115">كما يمكنك استكشاف وتحليل حركات دفتر الأستاذ العام من خلال تشغيل التقارير المالية.</span><span class="sxs-lookup"><span data-stu-id="2be02-115">You can also explore and analyze general ledger transactions by running financial reports.</span></span> <span data-ttu-id="2be02-116">ونظراً لأن تصميم التقارير المالية يمكنه أن يستند إلى الحسابات أو الأبعاد أو فئات الحساب أو مزيج من الثلاثة، يمكنك عرض الحركات من خلال التنقل لأسفل بطرق مختلفة.</span><span class="sxs-lookup"><span data-stu-id="2be02-116">Because the design of financial reports can be based on accounts, dimensions, account categories, or a combination of the three, you can view the transactions by drilling down in various ways.</span></span> <span data-ttu-id="2be02-117">وإذا كنت تحتاج إلى مزيد من المعلومات لحركات دفتر الأستاذ العام، يمكنك أيضًا تضمين عدة خصائص حركة كجزء من تصميم التقرير.</span><span class="sxs-lookup"><span data-stu-id="2be02-117">If you require more information for general ledger transactions, you can also include multiple transaction properties as part of the report design.</span></span> <span data-ttu-id="2be02-118">بالإضافة إلى ذلك، إذا كنت ترغب في الاطلاع على الحركات التي تشكل رصيد دفتر الأستاذ العام، فإنه يمكنك التنقل إلى حركات الحساب، تمامًا كما يمكنك القيام بذلك من صفحة قائمة **ميزان المراجعة**.</span><span class="sxs-lookup"><span data-stu-id="2be02-118">Additionally, if you want to see the transactions that make up a general ledger balance, you can drill down to account transactions, just as you can from the **Trial balance** list page.</span></span>

-   <span data-ttu-id="2be02-122">**قائمة حركات دفتر الأستاذ** – يعرض هذا التقرير الحركات في الحركة، والمحاسبة، وعملات التقارير لحساب.</span><span class="sxs-lookup"><span data-stu-id="2be02-122">**Ledger transaction list** – This report shows transactions in transaction, accounting, and reporting currencies for an account.</span></span>
-   <span data-ttu-id="2be02-123">**طباعة دفتر اليومية** - يبين هذا التقرير النتيجة لدفتر يومية تم ترحيله.</span><span class="sxs-lookup"><span data-stu-id="2be02-123">**Print journal** – This report shows the result of a posted journal.</span></span> <span data-ttu-id="2be02-124">ويمكنك تشغيل التقرير حسب نوع دفتر اليومية أو رقم دُفعة دفتر اليومية أو إضافة حقول إضافية.‬</span><span class="sxs-lookup"><span data-stu-id="2be02-124">You can run the report by journal batch number or journal type, or add additional fields.</span></span>
-   <span data-ttu-id="2be02-125">**الحركات المرحلة بواسطة دفتر اليومية‬** - يعرض هذا التقرير الحركات التي تم ترحيلها إلى دفتر يومية، وتم تجميعها حسب الإيصال.</span><span class="sxs-lookup"><span data-stu-id="2be02-125">**Posted transactions by journal** – This report shows the transactions that have been posted to a journal, grouped by voucher.</span></span>
-   <span data-ttu-id="2be02-126">**قائمة الحركات حسب التاريخ‬** - يعرض هذا التقرير كافة الحركات حسب التاريخ، جنبًا إلى جنب مع حساب دفتر الأستاذ ورقم دفتر اليومية والإيصال.</span><span class="sxs-lookup"><span data-stu-id="2be02-126">**Transaction list by date** – This report shows all the transactions by date, together with the journal number, voucher, and ledger account.</span></span> <span data-ttu-id="2be02-127">كما تُظهر الحركات في الحركة، والمحاسبة، وعملات التقارير.</span><span class="sxs-lookup"><span data-stu-id="2be02-127">It also shows the transactions in the transaction, accounting, and reporting currencies.</span></span>
-   <span data-ttu-id="2be02-128">**أصل الحركة** – يعرض تقرير الحركة هذا الحساب حسب دفتر اليومية والحركة والمحاسبة وعملة التقارير.</span><span class="sxs-lookup"><span data-stu-id="2be02-128">**Transaction origin** – This transaction report shows the account by journal, and by transaction, accounting, and reporting currency.</span></span> <span data-ttu-id="2be02-129">كما يُظهر كل بند دفتر يومية تم استخدامه كمقاصة.</span><span class="sxs-lookup"><span data-stu-id="2be02-129">It also shows each line of the journal that was used as an offset.</span></span>
