# <a name="cost-accounting-terminology"></a><span data-ttu-id="31650-103">مصطلحات محاسبة التكاليف</span><span class="sxs-lookup"><span data-stu-id="31650-103">Cost accounting terminology</span></span>

<span data-ttu-id="31650-104">يعرّف هذا الموضوع المصطلحات الأساسية المستخدمة في محاسبة التكاليف.</span><span class="sxs-lookup"><span data-stu-id="31650-104">This topic defines the key terms that are used in Cost accounting.</span></span>

<span data-ttu-id="31650-105">**أساس التوزيع**</span><span class="sxs-lookup"><span data-stu-id="31650-105">**Allocation base**</span></span>

<span data-ttu-id="31650-106">يُستخدم أساس التوزيع لقياس الأنشطة وحساب كميتها، مثل ساعات الأجهزة المستخدمة، أو ساعات الكيلووات (كيلووات) التي يتم استهلاكها، أو القدم المربع الذي يتم شغله.</span><span class="sxs-lookup"><span data-stu-id="31650-106">The allocation base is used to measure and quantify activities, such as machine hours that are used, kilowatt hours that are consumed, or square footage that is occupied.</span></span> <span data-ttu-id="31650-107">ويتم استخدامه كأساس لتوزيع التكاليف على كائن تكلفة واحد أو أكثر.</span><span class="sxs-lookup"><span data-stu-id="31650-107">It's used as basis for allocating costs to one or more cost objects</span></span>

<span data-ttu-id="31650-108">**محاسبة التكاليف**</span><span class="sxs-lookup"><span data-stu-id="31650-108">**Cost accounting**</span></span>

<span data-ttu-id="31650-109">تسمح لك محاسبة التكاليف بتجميع بيانات من مصادر مختلفة، مثل دفتر الأستاذ العام ودفاتر الأستاذ الفرعية والميزانيات والمعلومات الإحصائية.</span><span class="sxs-lookup"><span data-stu-id="31650-109">Cost accounting lets you collect data from various sources, such as the general ledger, sub-ledgers, budgets, and statistical information.</span></span> <span data-ttu-id="31650-110">ويمكنك عندئذٍ تحليل بيانات التكلفة وتلخيصها وتقييمها، لتمكين الإدارة من اتخاذ أفضل القرارات الممكنة لتحديثات السعر والميزانيات ومراقبة التكاليف وهكذا.</span><span class="sxs-lookup"><span data-stu-id="31650-110">You can then analyze, summarize, and evaluate cost data, so that management can make the best possible decisions for price updates, budgets, cost control, and so on.</span></span> <span data-ttu-id="31650-111">يتم التعامل مع البيانات المصدر المستخدمة لتحليل التكاليف بشكل مستقل في محاسبة التكاليف.</span><span class="sxs-lookup"><span data-stu-id="31650-111">The source data that is used for cost analysis is treated independently in Cost accounting.</span></span> <span data-ttu-id="31650-112">ونتيجة لذلك، لا تؤثر التحديثات في محاسبة التكاليف على البيانات المصدر.</span><span class="sxs-lookup"><span data-stu-id="31650-112">Therefore, updates in Cost accounting don’t affect the source data.</span></span> <span data-ttu-id="31650-113">ومع ذلك، عندما تقوم بتجميع بيانات التكلفة من مصادر مختلفة، وخاصة عند استيراد الحسابات الرئيسية من دفتر الأستاذ العام في Microsoft Dynamics 365 for Finance and Operations كعناصر تكلفة، هناك تكرار للبيانات، نظرًا لوجود البيانات نفسها في كل من دفتر الأستاذ العام ومحاسبة التكاليف.</span><span class="sxs-lookup"><span data-stu-id="31650-113">However, when you collect cost the same data exists in both General ledger and Cost accounting.</span></span> <span data-ttu-id="31650-114">هذا التكرار مطلوب، لأنك تستخدم الإدارة المالية لإعداد التقارير الخارجية ومحاسبة التكاليف لإعداد التقارير الداخلية.</span><span class="sxs-lookup"><span data-stu-id="31650-114">This redundancy is required, because you use financial management for external reporting and Cost accounting for <span data-ttu-id="31650-115">**دفتر أستاذ محاسبة التكاليف**</span><span class="sxs-lookup"><span data-stu-id="31650-115">**Cost accounting ledger**</span></span>

<span data-ttu-id="31650-116">إنه يتحكم بالعمليات والسياسات لقياس التكاليف، وهو محدد بواسطة أبعاد التقويم والعملة وعنصر التكلفة.</span><span class="sxs-lookup"><span data-stu-id="31650-116">Defined by calendar, currency, and cost element dimension, it controls processes and policies for measuring costs.</span></span> 

<span data-ttu-id="31650-117">**إدخال التكلفة**</span><span class="sxs-lookup"><span data-stu-id="31650-117">**Cost entry**</span></span>

<span data-ttu-id="31650-118">إدخالات التكلفة هي نتيجة لعملة تحويل عبر موصلات البيانات من إدخالات دفتر الأستاذ العام و التكلفة‬ وإدخالات التكلفة المرحلة في دفاتر يومية التكلفة.</span><span class="sxs-lookup"><span data-stu-id="31650-118">Cost entries are the result of a transfer via data connectors from general ledger entries, cost allocations, and posted cost entries in cost journals.</span></span>

<span data-ttu-id="31650-119">**كائن التكلفة**</span><span class="sxs-lookup"><span data-stu-id="31650-119">**Cost object**</span></span>

<span data-ttu-id="31650-120">أي نوع كائن تم تحديده للتحكم في التكاليف.</span><span class="sxs-lookup"><span data-stu-id="31650-120">Any type of object that is selected for cost control.</span></span> <span data-ttu-id="31650-121">يتم ترحيل التكاليف أو الإيرادات أو توزيعها بشكل مباشر على كائنات التكلفة.</span><span class="sxs-lookup"><span data-stu-id="31650-121">Costs or revenues are either directly posted on or allocated to cost objects.</span></span> <span data-ttu-id="31650-122">بعض كائنات التكلفة النموذجية هي:</span><span class="sxs-lookup"><span data-stu-id="31650-122">Some typical cost objects are:</span></span>

-  <span data-ttu-id="31650-123">المنتجات</span><span class="sxs-lookup"><span data-stu-id="31650-123">Products</span></span>
-  <span data-ttu-id="31650-124">المشاريع</span><span class="sxs-lookup"><span data-stu-id="31650-124">Projects</span></span>
-  <span data-ttu-id="31650-125">الأقسام</span><span class="sxs-lookup"><span data-stu-id="31650-125">Departments</span></span>
-  <span data-ttu-id="31650-126">مراكز تكلفة</span><span class="sxs-lookup"><span data-stu-id="31650-126">Cost centers</span></span>

<span data-ttu-id="31650-127">تستخدم الإدارة كائنات التكلفة لتحديد التكاليف، بل أيضًا لتعزيز تحليل الربحية.</span><span class="sxs-lookup"><span data-stu-id="31650-127">Management uses cost objects to quantify costs, but also to drive profitability analysis.</span></span>

<span data-ttu-id="31650-128">**عنصر التكلفة**</span><span class="sxs-lookup"><span data-stu-id="31650-128">**Cost element**</span></span>

<span data-ttu-id="31650-129">يستخدم كوظيفة لتعقب وتصنيف التكاليف.</span><span class="sxs-lookup"><span data-stu-id="31650-129">Used as a function to track and categorize costs.</span></span> <span data-ttu-id="31650-130">هناك نوعان من عناصر التكلفة: الأساسية والثانوية.</span><span class="sxs-lookup"><span data-stu-id="31650-130">There are two types of cost elements: primary and secondary.</span></span>

<span data-ttu-id="31650-131">تمثل عناصر التكلفة الأساسية تدفق التكاليف من المحاسبة المالية إلى محاسبة التكاليف.</span><span class="sxs-lookup"><span data-stu-id="31650-131">Primary cost elements represent the cost flow from financial accounting to Cost accounting.</span></span> <span data-ttu-id="31650-132">تتطابق بنية عنصر التكلفة عادةً مع بنية حساب الأرباح والخسائر في دفتر الأستاذ العام، حيث يمكن أن يتطابق عنصر تكلفة مع حساب رئيسي.</span><span class="sxs-lookup"><span data-stu-id="31650-132">The structure typically corresponds to the profit and loss account structure in the general ledger where a cost element can correspond to a main account.</span></span> <span data-ttu-id="31650-133">من غير الضروري أن تتمثل كافة الحسابات الرئيسية كعناصر تكلفة، وهذا يتوقف على متطلبات العمل.</span><span class="sxs-lookup"><span data-stu-id="31650-133">Not all main accounts must be represented as cost elements, depending on business requirements.</span></span> 

<span data-ttu-id="31650-134">تمثل عناصر التكلفة الثانوية التدفق الداخلي للتكاليف، إذ يتم استخدام هذه التكاليف فقط في محاسبة التكاليف.</span><span class="sxs-lookup"><span data-stu-id="31650-134">Secondary cost elements represent the internal cost flow because these costs are only used in Cost accounting.</span></span> <span data-ttu-id="31650-135">ويتم استخدامها في قواعد زيادة التكلفة لتجميع التكاليف في مستودعات ذات معنى تستخدمها عملية حساب المصروفات الزائدة.</span><span class="sxs-lookup"><span data-stu-id="31650-135">They are used in cost roll-up rules to aggregate costs into meaningful buckets used by overhead calculation.</span></span> 

<span data-ttu-id="31650-136">**تصنيف التكلفة**</span><span class="sxs-lookup"><span data-stu-id="31650-136">**Cost classification**</span></span>

<span data-ttu-id="31650-137">يقوم تصنيف التكلفة بجمع التكاليف وفقًا لخصائصها المشتركة.</span><span class="sxs-lookup"><span data-stu-id="31650-137">Cost classification groups costs according to their shared characteristics.</span></span> <span data-ttu-id="31650-138">على سبيل المثال، يمكن تجميع التكاليف حسب العناصر والتتبع والسلوك.</span><span class="sxs-lookup"><span data-stu-id="31650-138">For example, costs can be grouped by elements, traceability, and behavior.</span></span>

-   <span data-ttu-id="31650-139">**حسب العناصر** – المواد والعمالة والمصروفات.</span><span class="sxs-lookup"><span data-stu-id="31650-139">**By elements** – Materials, labor, and expenses.</span></span>
-   <span data-ttu-id="31650-140">**حسب التتبع** – التكاليف المباشرة وغير المباشرة.</span><span class="sxs-lookup"><span data-stu-id="31650-140">**By traceability** – Direct costs and indirect costs.</span></span> <span data-ttu-id="31650-141">يتم تعيين التكاليف المباشرة إلى كائنات التكلفة بشكل مباشر.</span><span class="sxs-lookup"><span data-stu-id="31650-141">Direct costs are assigned directly to cost objects.</span></span> <span data-ttu-id="31650-142">أما التكاليف غير المباشرة فلا يمكن تتعبها مباشرة إلى كائنات تكلفة.</span><span class="sxs-lookup"><span data-stu-id="31650-142">Indirect costs aren't directly traceable to cost objects.</span></span> <span data-ttu-id="31650-143">ويتم تخصيص التكاليف غير المباشرة لكائنات التكلفة.</span><span class="sxs-lookup"><span data-stu-id="31650-143">Indirect costs are allocated to cost objects.</span></span>
-   <span data-ttu-id="31650-144">**حسب السلوك** – الثابت والمتغير وشبه المتغير.</span><span class="sxs-lookup"><span data-stu-id="31650-144">**By behavior** – Fixed, variable, and semi-variable.</span></span>

<span data-ttu-id="31650-145">**سلوك التكلفة**</span><span class="sxs-lookup"><span data-stu-id="31650-145">**Cost behavior**</span></span>

<span data-ttu-id="31650-146">يصنف سلوك التكلفة التكاليف حسب سلوكها فيما يتعلق بالتغييرات في الأنشطة التجارية الرئيسية.</span><span class="sxs-lookup"><span data-stu-id="31650-146">Cost behavior classifies costs according to their behavior in relation to changes in key business activities.</span></span> <span data-ttu-id="31650-147">لمراقبة التكاليف بشكل فعال، يتعين على الإدارة فهم سلوك التكلفة.</span><span class="sxs-lookup"><span data-stu-id="31650-147">To control costs effectively, management must understand the cost behavior.</span></span> <span data-ttu-id="31650-148">هناك ثلاثة أنواع من نمط سلوك التكلفة: الثابت والمتغير وشبه المتغير.</span><span class="sxs-lookup"><span data-stu-id="31650-148">There are three types of cost behavior pattern: fixed, variable, and semi-variable.</span></span>

- <span data-ttu-id="31650-149">**التكلفة الثابتة** - التكلفة الثابتة هي تكلفة لا تتغير على المدى القصير، بغض النظر عن التغييرات في مستوى النشاط.</span><span class="sxs-lookup"><span data-stu-id="31650-149">**Fixed cost** - A fixed cost is a cost that doesn't vary in the short term, regardless of changes in activity level.</span></span> <span data-ttu-id="31650-150">على سبيل المثال، بإمكان التكلفة الثابتة أن تكون عبارة عن مصروفات تشغيل أساسية لشركة ما، مثل الإيجار، لن تتأثر حتى عند ازدياد مستوى النشاط أو انخفاضه.</span><span class="sxs-lookup"><span data-stu-id="31650-150">For example, a fixed cost can be a basic operating expense of a business, such as rent, that won't be affected even if the activity level increases or decreases.</span></span>

- <span data-ttu-id="31650-151">**التكلفة المتغيرة‬** - تتغير التكلفة المتغيرة عند حدوث تغييرات في مستوى النشاط.</span><span class="sxs-lookup"><span data-stu-id="31650-151">**Variable cost** - A variable cost changes according to changes in activity level.</span></span> <span data-ttu-id="31650-152">على سبيل المثال، ترتبط تكلفة مواد مباشرة محددة بكل منتج يتم بيعه.</span><span class="sxs-lookup"><span data-stu-id="31650-152">For example, a specific direct materials cost is associated with each product that is sold.</span></span> <span data-ttu-id="31650-153">بقدر ما يزيد عدد المنتجات المباعة، تزداد تكاليف المواد المباشرة.</span><span class="sxs-lookup"><span data-stu-id="31650-153">The more products that are sold, the more direct materials costs there are.</span></span>

- <span data-ttu-id="31650-154">**التكلفة شبه المتغيرة‬** - التكاليف شبه المتغيرة هي تكاليف ثابتة جزئيًا ومتغيرة جزئيًا.</span><span class="sxs-lookup"><span data-stu-id="31650-154">**Semi-variable cost** - Semi-variable costs are partly fixed and partly variable costs.</span></span> <span data-ttu-id="31650-155">على سبيل المثال، تتضمن تكلفة الوصول إلى الإنترنت رسم وصول شهري قياسي ورسم استخدام النطاق الترددي الواسع.</span><span class="sxs-lookup"><span data-stu-id="31650-155">For example, an Internet access fee includes a standard monthly access fee and a broadband usage fee.</span></span> <span data-ttu-id="31650-156">يُعد رسم الوصول الشهري القياسي تكلفة ثابتة، في حين يُعد رسم استخدام النطاق الترددي الواسع تكلفة متغيرة.</span><span class="sxs-lookup"><span data-stu-id="31650-156">The standard monthly access fee is a fixed cost, whereas the broadband usage fee is a variable cost.</span></span>

<span data-ttu-id="31650-157">**وحدة التحكم بالتكاليف**</span><span class="sxs-lookup"><span data-stu-id="31650-157">**Cost control unit**</span></span>

<span data-ttu-id="31650-158">تمثل وحدة التحكم بالتكاليف بنية التكلفة.</span><span class="sxs-lookup"><span data-stu-id="31650-158">The cost control unit represents the cost structure.</span></span> <span data-ttu-id="31650-159">تحدد البنية كيف تتدفق التكلفة بترتيب تدرج هرمي بين أبعاد كائنات التكلفة وكائنات التكلفة الخاصة بها.</span><span class="sxs-lookup"><span data-stu-id="31650-159">The structure determines how cost flows in a hierarchical order between cost object dimensions and their respective cost objects.</span></span> 

<span data-ttu-id="31650-160">**توزيع التكلفة**</span><span class="sxs-lookup"><span data-stu-id="31650-160">**Cost distribution**</span></span>

<span data-ttu-id="31650-161">يتم استخدام توزيع التكلفة لتوزيع التكلفة من كائن تكلفة واحد إلى كائن تكلفة واحد أو أكثر عن طريق تطبيق أساس توزيع ذي صلة.</span><span class="sxs-lookup"><span data-stu-id="31650-161">Is used to distribute cost from one cost object to one or more other cost objects by applying a relevant allocation base.</span></span> <span data-ttu-id="31650-162">هناك اختلاف بين توزيع التكلفة وتخصيص التكلفة، حيث يحدث توزيع التكلفة دائمًا على مستوى عنصر التكلفة الأساسية‬‏‫ للتكلفة الأصلية وبدون معالجة متبادلة.</span><span class="sxs-lookup"><span data-stu-id="31650-162">Cost distribution and cost allocation differ in that cost distribution always occurs at the level of the primary cost element of the original cost and no reciprocal processing.</span></span>

<span data-ttu-id="31650-163">**توزيع التكلفة**</span><span class="sxs-lookup"><span data-stu-id="31650-163">**Cost allocation**</span></span>

<span data-ttu-id="31650-164">يُستخدم لتخصيص رصيد كائن التكلفة إلى كائنات تكلفة أخرى عن طريق تطبيق أساس التوزيع.</span><span class="sxs-lookup"><span data-stu-id="31650-164">Is used to allocate the balance of a cost object to other cost objects by applying an allocation base.</span></span> <span data-ttu-id="31650-165">يدعم تطبيق Finance and Operations طريقة التوزيع المتبادل.</span><span class="sxs-lookup"><span data-stu-id="31650-165">Finance and Operations supports the reciprocal allocation method.</span></span> <span data-ttu-id="31650-166">في أسلوب التخصيص المتبادل، يتم التعرّف بشكل كامل على الخدمات المتبادلة التي تتبادلها كائنات التكلفة المساعدة.</span><span class="sxs-lookup"><span data-stu-id="31650-166">In the reciprocal allocation method, the mutual services that auxiliary cost objects exchange are fully recognized.</span></span> <span data-ttu-id="31650-167">يحدد النظام بشكل تلقائي الترتيب الصحيح لتنفيذ عمليات التخصيص فيه وتكرارها.</span><span class="sxs-lookup"><span data-stu-id="31650-167">The system automatically determines the order to perform the allocations in and iterate over it.</span></span> <span data-ttu-id="31650-168">يتم تخصيص الرصيد الخاص بكائن التكلفة بواسطة أساس توزيع فردي.</span><span class="sxs-lookup"><span data-stu-id="31650-168">The balance of a cost object is allocated by a single allocation base.</span></span> <span data-ttu-id="31650-169">يتم دعم عمليات التخصيص عبر أبعاد كائنات التكلفة وأعضائها.</span><span class="sxs-lookup"><span data-stu-id="31650-169">Allocations across cost object dimensions and their respective members are supported.</span></span> <span data-ttu-id="31650-170">يتم التحكم بترتيب التخصيص بواسطة وحدة التحكم في التكلفة.</span><span class="sxs-lookup"><span data-stu-id="31650-170">The allocation order is controlled by the cost control unit.</span></span>

<span data-ttu-id="31650-171">**سياسة توزيع التكلفة**</span><span class="sxs-lookup"><span data-stu-id="31650-171">**Cost allocation policy**</span></span>
