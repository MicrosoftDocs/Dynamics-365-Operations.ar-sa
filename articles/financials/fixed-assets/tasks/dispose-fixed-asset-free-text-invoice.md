[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="0215e-104">يشرح هذا الموضوع كيفية التخلص من أصل ثابت باستخدام فاتورة نص حر.</span><span class="sxs-lookup"><span data-stu-id="0215e-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="0215e-105">في جزء التنقل، انتقل إلى **الوحدات النمطية > الحسابات المدينة > الفواتير > جميع فواتير النص الحر**.</span><span class="sxs-lookup"><span data-stu-id="0215e-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="0215e-106">حدد **جديد**.</span><span class="sxs-lookup"><span data-stu-id="0215e-106">Select **New**.</span></span>
3. <span data-ttu-id="0215e-107">أدخل قيمة أو حددها في حقل **حساب العميل**.</span><span class="sxs-lookup"><span data-stu-id="0215e-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="0215e-108">تحقق من صحة تاريخ **الفاتورة** الافتراضي وقم بتحريره إذا كان ذلك ممكنًا.</span><span class="sxs-lookup"><span data-stu-id="0215e-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="0215e-109">تحقق من صحة حقول الرأس الافتراضية المتبقية، مثل **العملة** وقم بتحريرها إذا كان ذلك ممكنًا.</span><span class="sxs-lookup"><span data-stu-id="0215e-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="0215e-110">في القسم **بنود الفاتورة**، تحقق من صحة حقلي **الوصف** و**الحساب الرئيسي**.</span><span class="sxs-lookup"><span data-stu-id="0215e-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="0215e-111">تحقق من صحة الحقلين **مجموعة ضريبة المبيعات الافتراضية** و**مجموعة ضريبة مبيعات الصنف**.</span><span class="sxs-lookup"><span data-stu-id="0215e-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="0215e-112">أدخل **سعر الوحدة** أو **مبلغ** عملية بيع الأصل الثابت.</span><span class="sxs-lookup"><span data-stu-id="0215e-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="0215e-113">حدد قسم **تفاصيل البنود**.</span><span class="sxs-lookup"><span data-stu-id="0215e-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="0215e-114">في حقل **الأصل الثابت**، أدخل قيمة أو حددها.</span><span class="sxs-lookup"><span data-stu-id="0215e-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="0215e-115">حدد **ترحيل**.</span><span class="sxs-lookup"><span data-stu-id="0215e-115">Select **Post**.</span></span>
