[!include[banner](../includes/banner.md)]

<span data-ttu-id="e6e0c-104">يمكنك تتبع نتائج تنسيقات إعداد التقارير الإكترونية التي تعمل على إنشاء مستندات إلكترونية صادرة.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-104">You can trace the results of ER formats that generate outgoing electronic documents.</span></span> <span data-ttu-id="e6e0c-105">عندما يتم تشغيل إنشاء تتبع (تشغيل معلمة مستخدم إعداد التقارير الإكترونية **في وضع التصحيح**)، في سجل تنفيذ بتنسيق إعداد التقارير الإكترونية في كل مرة يتم فيها تشغيل تقرير إعداد التقارير الإكترونية.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-105">When trace generation is turned on (ER user parameter **Run in debug mode**), a new trace record is generated in the ER format execution log every time that an ER report is run.</span></span> <span data-ttu-id="e6e0c-106">يتم تخزين البيانات التالية في كل تتبع يتم إنشاؤه:</span><span class="sxs-lookup"><span data-stu-id="e6e0c-106">The following details are stored in each trace that is generated:</span></span>

- <span data-ttu-id="e6e0c-108">كافة الأخطاء التي تم إنشاؤها بواسطة قواعد التحقق</span><span class="sxs-lookup"><span data-stu-id="e6e0c-108">All errors that were generated by validation rules</span></span>
- <span data-ttu-id="e6e0c-109">كافة الملفات المنشأة التي يتم تخزينها كمرفقات لسجل التتبع</span><span class="sxs-lookup"><span data-stu-id="e6e0c-109">All generated files that are stored as attachments of the trace record</span></span>

<span data-ttu-id="e6e0c-110">يمكنك تخزين ملفات تطبيق خط أساسي فردي لأي تنسيق إعداد التقارير الإلكترونية.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-110">You can store individual baseline application files for any ER format.</span></span> <span data-ttu-id="e6e0c-111">تُعتبر الملفات ملفات خط أساسي عندما تصف النتائج المتوقعة للتقارير التي تكون قيد التشغيل.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-111">Files are considered baseline files when they describe the expected results of reports that are run.</span></span> <span data-ttu-id="e6e0c-112">إذا كان ملف الخط الأساسي متوافر لتنسيق إعداد التقارير الإكترونية تم تشغيله أثناء تشغيل إنشاء تتبع، يُخزن التتبع، بالإضافة إلى التفاصيل التي كانت مذكورة سابقًا، نتيجة مقارنة المستند الإلكتروني المنشأ على ملف الخط الأساسي.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-112">If a baseline file is available for an ER format that was run while trace generation was turned on, the trace stores, in addition to the details that were mentioned earlier, the result of the comparison of the generated electronic document to the baseline file.</span></span> <span data-ttu-id="e6e0c-113">بنقرة واحدة، يمكنك كذلك الحصول على الملف الإلكتروني المنشأ وملف الخط الأساسي الخاص به في ملف واحد مضغوط.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-113">In one click, you can also get the generated electronic document and its baseline file in a single zip file.</span></span> <span data-ttu-id="e6e0c-114">يمكنك حينها القيام بمقارنة مفصلة باستخدام أداة خارجية مثل Windiff.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-114">You can then do detailed comparison by using an external tool such as Windiff.</span></span>

<span data-ttu-id="e6e0c-115">يمكنك تقييم التتبع لتحليل ما إذا كانت المستندات الإلكترونية التي يتم إنشاؤها تتضمن المحتوى المتوقع أم لا.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-115">You can evaluate the trace to analyze whether the electronic documents that are generated include the expected content.</span></span> <span data-ttu-id="e6e0c-116">يمكنك القيام بهذا التقييم في بيئة اختبار قبول المستخدم (UAT) عندما تم تغيير قاعدة الترميز (على سبيل المثال، عند الترحيل إلى مثيل جديد للتطبيق أو تثبيت حزم إصلاح عاجل أو نشر تعديلات الكود).</span><span class="sxs-lookup"><span data-stu-id="e6e0c-116">You can do this evaluation in a user acceptance testing (UAT) environment when the code base has been changed (for example, when you migrated to a new instance of the application, installed hotfix packages, or deployed code modifications).</span></span> <span data-ttu-id="e6e0c-117">بهذه الطريقة، يمكنك التأكد من أن التقييم لا يؤثر على تنفيذ تقارير إعداد التقارير الإلكترونية التي يتم استخدامها.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-117">In this way, you can make sure that the evaluation doesn't affect the execution of ER reports that are used.</span></span> <span data-ttu-id="e6e0c-118">بالنسبة للعديد من تقارير إعداد التقارير الإكترونية، فإنه يمكن القيام بالتقييم وضع غير مراقب.</span><span class="sxs-lookup"><span data-stu-id="e6e0c-118">For many ER reports, the evaluation can be done in unattended mode.</span></span>
