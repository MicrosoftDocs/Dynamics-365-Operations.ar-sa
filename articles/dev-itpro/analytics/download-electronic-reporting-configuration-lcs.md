<span data-ttu-id="09334-105">يرشدك هذا البرنامج التعليمي عبر عملية تنزيل الإصدار الأحدث من تكوينات التقارير الإلكترونية من Microsoft Dynamics Lifecycle Services (LCS).</span><span class="sxs-lookup"><span data-stu-id="09334-105">This tutorial guides you through the process of downloading the newest version of Electronic reporting (ER) configurations from Microsoft Dynamics Lifecycle Services (LCS).</span></span>

    1. <span data-ttu-id="09334-116">انقر فوق **إضافة** لإضافة مستود جديد.</span><span class="sxs-lookup"><span data-stu-id="09334-116">Click **Add** to add a new repository.</span></span>
    2. <span data-ttu-id="09334-117">حدد **LCS** كنوع المستودع.</span><span class="sxs-lookup"><span data-stu-id="09334-117">Select **LCS** as the repository type.</span></span>
    3. <span data-ttu-id="09334-118">انقر فوق **إنشاء مستودع**.</span><span class="sxs-lookup"><span data-stu-id="09334-118">Click **Create repository**.</span></span>
    4. <span data-ttu-id="09334-119">اتبع الإرشادات التخويل، إذا طلب منك ذلك.</span><span class="sxs-lookup"><span data-stu-id="09334-119">If prompted, follow the authorization instructions.</span></span>

    7. <span data-ttu-id="09334-122">في الشبكة، حدد المستودع الجديد من النوع **LCS‎**.</span><span class="sxs-lookup"><span data-stu-id="09334-122">In the grid, select the new repository of the **LCS** type.</span></span>

    <span data-ttu-id="09334-124">[![تحديث التقارير الإلكترونية من LCS لـ MS - عمل ‏‫مستودع LCS](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)</span><span class="sxs-lookup"><span data-stu-id="09334-124">[![update-er-from-lcs-for-ms-make-lcs-repository](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)</span></span>

7. <span data-ttu-id="09334-125">في شجرة التكوينات في الجزء الأيمن، حدد تكوين التقارير الإلكترونية المطلوب.</span><span class="sxs-lookup"><span data-stu-id="09334-125">In the configurations tree in the left pane, select the ER configuration that you require.</span></span>
8. <span data-ttu-id="09334-126">على علامة التبويب السريعة **إصدارات**، حدد الإصدار المطلوب لتكوين التقارير الإلكترونية المحدد.</span><span class="sxs-lookup"><span data-stu-id="09334-126">On the **Versions** FastTab, select the required version of the selected ER configuration.</span></span>

    > [!NOTE]
    > <span data-ttu-id="09334-128">لا يتوفر الزر **استيراد**‏‎ لإصدارات تكوينات التقارير الإلكترونية الموجودة في مثيل Finance and Operations الحالي.</span><span class="sxs-lookup"><span data-stu-id="09334-128">The **Import** button is unavailable for ER configuration versions that are already present in the current Finance and Operations instance.</span></span>
